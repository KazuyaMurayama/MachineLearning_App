---

## 評価フレームワーク

### 評価軸（6軸 × 各1-5点 = 合計30点満点）

| 評価軸 | 略称 | 観点 |
|--------|------|------|
| **市場規模** | 市場 | 対象企業数 × 潜在的な1社あたり売上（5=数万社・数十億円規模、1=極めてニッチ） |
| **ペインの深さ** | ペイン | 課題の深刻度と業績インパクト（5=年間数千万〜億円、1=課題認識なし） |
| **データ入手性** | データ | 顧客がCSV/Excelで教師データを準備できるか（5=既に管理済み、1=データなし） |
| **競合優位性** | 競合 | 既存ソリューションとの差別化余地（5=代替なし、1=レッドオーシャン） |
| **課金意欲** | 課金 | ターゲットの支払い意思（5=年間数百万円以上支出済み、1=IT投資理解なし） |
| **アプリ適合度** | 適合 | LightGBM回帰/ノーコード/SHAPとの相性（5=完璧フィット、1=対応困難） |

### スコアリング手法
- 5つの専門エージェントチーム（製造・物流 / 小売・EC・飲食 / 金融・不動産・保険 / HR・医療・教育・SaaS / マーケ・広告・コンサル・ニッチ）が独立評価
- 業界知見＋ビジネス分析の両面からスコアリング
- 各チームがTop 5を選出し、全体統合ランキングを作成
