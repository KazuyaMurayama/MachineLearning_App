# 料金およびスコープ定義書（Pricing & Scope）

本書は、MachineLearning_App（以下「本サービス」）の料金体系、プラン構成、提供範囲、SLA（サービスレベル合意）の概要を一元管理する唯一の情報源（Single Source of Truth）です。提案書・契約書・見積書を作成する際は、必ず本書の最新バージョンと整合していることを確認してください。

## 1. プラン概要

本サービスは、士業（税理士・会計士事務所）および中堅EC事業者向けに、3階層のプラン構成で提供しています。

| プラン | 対象フェーズ | 期間 | 主用途 |
| --- | --- | --- | --- |
| パイロット | 導入検証 | 6週間固定 | L3アプリのPoC、効果測定、継続判断 |
| スタンダード | 本番運用 | 12か月契約（自動更新） | 日常業務への組み込み、月次KPI運用 |
| エンタープライズ | 全社展開 | 12か月契約（要個別設計） | 複数拠点、カスタムモデル、専任体制 |

## 2. 業種別定価

### 2.1 士業向け（税理士・会計士事務所）

| プラン | 定価（税別） | 提供アプリ |
| --- | --- | --- |
| パイロット | ¥150,000 / 6週間 | 事務所ダッシュボード、LTV予測、AIブリーフィング |
| スタンダード | ¥650,000 / 月 | 上記3アプリ＋月次レポート＋サポート |
| エンタープライズ | 個別見積 | 上記＋カスタムモデル＋専任CS |

パイロット価格¥150,000はスタンダード通常価格¥650,000に対して約77%割引に相当します。割引はパイロット期間の1回限り適用です。

### 2.2 EC向け（中堅EC事業者）

| プラン | 定価（税別） | 提供アプリ |
| --- | --- | --- |
| パイロット | ¥120,000 / 6週間 | 経営ダッシュボード、What-Ifシミュレーター、AIブリーフィング |
| スタンダード | ¥530,000 / 月 | 上記3アプリ＋月次レポート＋サポート |
| エンタープライズ | 個別見積 | 上記＋カスタムモデル＋専任CS |

パイロット価格¥120,000はスタンダード通常価格¥530,000に対して約77%割引に相当します。

> 注記: EC向けWhat-Ifシミュレーターに含まれる予測モデル `ec-what-if` は、現在精度の再検証中です（内部検証値 R²=-0.16）。パイロット期間中は参考値として提示し、確定した予測値としての提示は行いません。本件の状況はスタンダード移行時の精度基準（後述）に連動します。

## 3. 料金に含まれるもの・含まれないもの

### 3.1 パイロットプランに含まれるもの

- L3アプリ（業種別3アプリ）のホステッド環境提供
- 初期データ取り込み支援（CSV / API、最大2データソース）
- キックオフミーティング（1回、90分）
- 週次進捗ミーティング（6回、各45分）
- 中間レビュー（1回、60分）
- 最終レビューと継続判断ミーティング（1回、90分）
- AIブリーフィングの初期プロンプト調整（2回まで）
- 平日9:00〜18:00のメールサポート

### 3.2 パイロットプランに含まれないもの

- 顧客固有データを用いた予測モデルの再学習・カスタマイズ
- 基幹システムとのAPI連携開発
- オンプレミス環境へのデプロイ
- 24時間365日のサポート
- 訪問対応(オンサイト)
- セキュリティ監査対応資料の個別作成（簡易版はスタンダード以降で提供）

不明確な境界が発生した場合は、Sales Opsに確認のうえ、見積段階で「スコープ外」または「追加オプション」として明示してください。

## 5. SLA概要

本SLAはスタンダード以降に適用されます。パイロットプランは「ベストエフォート提供」とし、以下の目標値に準ずる努力を行いますが、法的なSLA保証の対象外です。

| 項目 | 目標値 |
| --- | --- |
| 月間稼働率 | 99.5%以上（仮目標、要検証） |
| 重大障害時の一次応答 | 2営業時間以内 |
| 通常問い合わせの一次応答 | 1営業日以内 |
| サポートチャネル | メール、チャット（平日9:00〜18:00） |
| 計画メンテナンス窓 | 毎月第2日曜 01:00〜05:00（JST） |
| メンテナンス事前通知 | 5営業日前まで |

予測モデルの精度に関するSLAは現時点では設定していません。精度目標は各モデルのリリースノートに記載します。

## 6. 支払条件・請求サイクル

- パイロット: 契約締結後、開始前に全額請求（前払い）。振込期限は請求書発行日より14日以内。
- スタンダード: 月次請求、翌月末払い。
- エンタープライズ: 四半期または年次の前払いを推奨。個別交渉可。
- 支払方法: 銀行振込（原則）。クレジットカード決済は準備中。
- 通貨: 日本円（JPY）のみ。
- 消費税は別途申し受けます。

## 7. 値引き承認ルール

定価からの追加値引きには以下の承認ルールを適用します。

| 値引き率 | 承認者 | 必要書類 |
| --- | --- | --- |
| 〜5% | 営業担当 | 見積書（記録） |
| 〜15% | 営業マネージャ | 見積書＋値引き理由書 |
| 〜30% | 事業責任者 | 上記＋案件概要シート |
| 30%超 | 役員 | 上記＋経営承認稟議 |

パイロットプランの¥150,000 / ¥120,000は既に約77%割引済みのため、さらなる値引きは原則行いません。例外対応は役員承認を必須とします。
