# ROIカリキュレーター・ワークシート

本書は、商談のクロージングフェーズにおいて、顧客と一緒に埋めながら本サービス導入のROI（投資対効果）を可視化するためのワークシートです。顧客の実データを用いて数値を埋めることを前提とし、提案段階で仮置きする数値には必ず「仮説」「要検証」の注記を付けてください。

## 1. 基本方針

- ROIは「粗利改善効果 + 時間削減の金銭換算 − 本サービス利用料」で算出します。
- 数値は必ず「保守的 / 標準 / 楽観的」の3シナリオで提示します。
- 予測モデルを根拠とする場合、モデルの精度検証状況を必ず併記します。
- 本ワークシートは意思決定の補助資料であり、将来の成果を保証するものではありません。
- **まずパイロット期間ROI（6週間）を提示し、納得の上で年間ROIを議論する流れを推奨します。**

## 2. 基本式

```
■ パイロット期間ROI（推奨: まずこちらを提示）
パイロットROI = (年間効果額 × 6/52) − パイロット費用

■ 年間ROI（スタンダード移行検討時）
年間ROI金額 = (粗利改善額 + 時間削減金額) − 年間利用料
ROI率 (%) = 年間ROI金額 ÷ 年間利用料 × 100
```

## 3. 士業版

### 3.1 入力項目

| 項目 | 記号 | 単位 | 備考 |
| --- | --- | --- | --- |
| 顧問先数 | A | 件 | 現在の契約件数 |
| 平均月額顧問料 | B | 円 | 過去12か月平均 |
| 年間離反率（現状） | C | % | 解約件数 ÷ 年初件数 |
| 年間離反率（導入後仮説） | C' | % | LTV予測活用後の目標値 |
| 1件離反時の年間損失 | D | 円 | B × 12 |
| スタッフ時給 | E | 円 | 所長含め加重平均 |
| 月間削減時間仮説 | F | 時間 | ダッシュボードによる工数削減 |
| 新規受注件数（年間） | G | 件 | AIブリーフィング活用想定 |
| 新規1件あたりの年間売上 | H | 円 | 平均値 |

### 3.2 計算式

```
粗利改善額 = (C − C') × A × D + G × H × 粗利率
時間削減金額 = F × 12 × E
```

### 3.3 計算例（士業）

前提（すべて仮説・要顧客検証）:

- 顧問先数 A = 100件
- 平均月額顧問料 B = ¥30,000
- 現状離反率 C = 6%、導入後離反率 C' = 3%（🔸仮説）
- 1件離反時年間損失 D = ¥360,000
- スタッフ時給 E = ¥4,000、月間削減時間 F = 20時間（🔸仮説）
- 新規受注 G = 3件/年、H = ¥360,000、粗利率 70%（🔸仮説）

**パイロット期間ROI（6週間）— まずこちらを提示:**

- 年間効果額合計 = ¥2,796,000
- 6週間按分効果 = ¥2,796,000 × 6/52 = 約¥323,000
- パイロット費用 = ¥100,000
- **パイロットROI = +¥223,000（黒字）**

→ 6週間¥100,000の投資でこれだけの効果仮説が検証できる、という提示をしてください。

**年間ROI（スタンダード移行検討時）:**

- 年間効果額合計 = ¥2,796,000
- 年間利用料（スタンダード）= ¥200,000 × 12 = ¥2,400,000
- 年間ROI = +¥396,000（標準シナリオで黒字）
- **回収成立条件**: 標準シナリオで黒字化。保守的シナリオでも離反率改善2pt以上で回収可能

> 営業上の注意: パイロットROI・年間ROIともに黒字です。スタンダード移行の提案に使えます。

## 4. EC版

### 4.1 入力項目

| 項目 | 記号 | 単位 | 備考 |
| --- | --- | --- | --- |
| 月商 | A | 円 | 直近3か月平均 |
| 粗利率 | B | % | 原価控除後 |
| 月間広告費 | C | 円 | 全チャネル合計 |
| 施策改善仮説（粗利改善率） | H | % | 導入後の粗利率改善 |
| 施策改善仮説（広告効率改善率） | I | % | CPA削減率 |

### 4.2 計算例（EC）

前提（すべて仮説・要顧客検証）:

- 月商 A = ¥50,000,000、粗利率 B = 30%
- 月間広告費 C = ¥8,000,000
- 粗利改善仮説 H = 1.5%（🔸仮説）、広告効率改善仮説 I = 5%（🔸仮説）

**パイロット期間ROI（6週間）:**

- 年間効果額合計 = ¥7,500,000
- 6週間按分効果 = ¥7,500,000 × 6/52 = 約¥865,000
- パイロット費用 = ¥125,000
- **パイロットROI = +¥740,000（黒字）**

**年間ROI（スタンダード移行検討時）:**

- 年間効果額合計 = ¥7,500,000
- 年間利用料 = ¥250,000 × 12 = ¥3,000,000
- **年間ROI = +¥4,500,000、ROI率 150%**

> 重要注記: 上記の改善仮説（H、I）は `ec-what-if` モデル（R²=-0.16、精度再検証中）に基づくものではなく、業界一般的なベンチマーク仮説です。ROI提示には必ず顧客の過去実績に基づく保守的な仮説を併用してください。

## 5. 3シナリオ提示フォーマット

商談時は以下のテーブルを埋めて顧客に提示してください。

| シナリオ | 前提 | 6週間効果（パイロット） | パイロット費用 | パイロットROI | 年間効果 | 年間利用料 | 年間ROI |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 保守的 | 改善率を半分に割引 | | | | | | |
| 標準 | 仮説通り | | | | | | |
| 楽観的 | 改善率を1.5倍 | | | | | | |

**記入例（EC標準シナリオ）**: 6週間効果¥865k / パイロット¥125k / ROI+¥740k / 年間効果¥7.5M / 年間¥3.0M / ROI+¥4.5M

保守的シナリオのパイロットROIが黒字にならない場合は、提案構成の見直しを検討。

## 6. 注意書きと免責条項

- 本ワークシートの数値は顧客の実データで必ず再計算してください。
- `ec-what-if` 予測モデルは精度再検証中のため、ROI根拠には使用しないでください。
- 本書は将来の業績を保証するものではありません。
- ROI試算結果は意思決定の一材料であり、最終投資判断は顧客の責任で行われるものとします。
- 料金は `pricing-and-scope.md`（SSOT）に準拠。本書の金額と不一致がある場合はSSOTが正です。
