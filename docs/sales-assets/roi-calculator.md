# ROIカリキュレーター・ワークシート

本書は、商談のクロージングフェーズにおいて、顧客と一緒に埋めながら本サービス導入のROI（投資対効果）を可視化するためのワークシートです。顧客の実データを用いて数値を埋めることを前提とし、提案段階で仮置きする数値には必ず「仮説」「要検証」の注記を付けてください。

## 1. 基本方針

- ROIは「粗利改善効果 + 時間削減の金銭換算 − 本サービス利用料」で算出します。
- 数値は必ず「保守的 / 標準 / 楽観的」の3シナリオで提示します。
- 顧客の実データが得られていない段階では、業界平均や仮説値を使い、その旨を明示します。
- 予測モデルを根拠とする場合、モデルの精度検証状況を必ず併記します。
- 本ワークシートは意思決定の補助資料であり、将来の成果を保証するものではありません。

## 2. 基本式

```
年間ROI金額 = (粗利改善額 + 時間削減金額) − 年間利用料
ROI率 (%) = 年間ROI金額 ÷ 年間利用料 × 100
回収期間 (月) = 年間利用料 ÷ ((粗利改善額 + 時間削減金額) ÷ 12)
```

## 3. 士業版 入力項目表

### 3.1 入力項目

| 項目 | 記号 | 単位 | 備考 |
| --- | --- | --- | --- |
| 顧問先数 | A | 件 | 現在の契約件数 |
| 平均月額顧問料 | B | 円 | 過去12か月平均 |
| 年間離反率（現状） | C | % | 解約件数 ÷ 年初件数 |
| 年間離反率（導入後仮説） | C' | % | LTV予測活用後の目標値 |
| 1件離反時の年間損失 | D | 円 | B × 12 |
| スタッフ時給 | E | 円 | 所長含め加重平均 |
| 月間削減時間仮説 | F | 時間 | ダッシュボードによる工数削減 |
| 新規受注件数（年間） | G | 件 | AIブリーフィング活用想定 |
| 新規1件あたりの年間売上 | H | 円 | 平均値 |

### 3.2 計算式

```
粗利改善額 = (C − C') × A × D + G × H × 粗利率
時間削減金額 = F × 12 × E
```

### 3.3 計算例（士業）

前提（すべて仮説・要顧客検証）:

- 顧問先数 A = 100件
- 平均月額顧問料 B = ¥30,000
- 現状離反率 C = 6%
- 導入後離反率 C' = 3%（LTV予測による早期介入の効果仮説）
- 1件離反時年間損失 D = ¥360,000
- スタッフ時給 E = ¥4,000
- 月間削減時間 F = 20時間
- 新規受注 G = 3件 / 年、H = ¥360,000、粗利率 70%

計算:

- 粗利改善額（離反抑制）= (0.06 − 0.03) × 100 × 360,000 = ¥1,080,000
- 粗利改善額（新規受注）= 3 × 360,000 × 0.7 = ¥756,000
- 時間削減金額 = 20 × 12 × 4,000 = ¥960,000
- 合計効果 = ¥2,796,000
- 年間利用料（スタンダード想定）= ¥650,000 × 12 = ¥7,800,000

この例では単年では投資回収に至りません。顧問先数が多い事務所や、離反抑制効果が大きい場合にROIが成立しやすい構造です。実データで再計算してください。

## 4. EC版 入力項目表

### 4.1 入力項目

| 項目 | 記号 | 単位 | 備考 |
| --- | --- | --- | --- |
| 月商 | A | 円 | 直近3か月平均 |
| 粗利率 | B | % | 原価控除後 |
| 月間広告費 | C | 円 | 全チャネル合計 |
| 現状CPA | D | 円 | 新規獲得コスト |
| 現状CVR | E | % | サイト全体平均 |
| 平均LTV | F | 円 | 12か月累計 |
| 月次解約率（サブスク） | G | % | 該当する場合 |
| 施策改善仮説（粗利改善率） | H | % | 導入後の粗利率改善 |
| 施策改善仮説（広告効率改善率） | I | % | CPA削減率 |

### 4.2 計算式

```
粗利改善額（年間）= A × 12 × B × H
広告費削減額（年間）= C × 12 × I
時間削減金額 = (月間削減時間) × 12 × (担当時給)
```

### 4.3 計算例（EC）

前提（すべて仮説・要顧客検証）:

- 月商 A = ¥50,000,000
- 粗利率 B = 30%
- 月間広告費 C = ¥8,000,000
- 粗利改善仮説 H = 1.5%
- 広告効率改善仮説 I = 5%

計算:

- 粗利改善額 = 50,000,000 × 12 × 0.30 × 0.015 = ¥2,700,000
- 広告費削減額 = 8,000,000 × 12 × 0.05 = ¥4,800,000
- 合計効果 = ¥7,500,000
- 年間利用料（スタンダード想定）= ¥530,000 × 12 = ¥6,360,000
- 年間ROI金額 = ¥1,140,000、ROI率 約17.9%

> 重要注記: 上記の改善仮説（H、I）の一部はWhat-Ifシミュレーターの予測結果を参照することがありますが、現時点で `ec-what-if` モデルは R²=-0.16 と精度再検証中の状態です。シミュレーター出力は参考値として扱い、ROI提示には必ず顧客の過去実績に基づく保守的な仮説を併用してください。

## 5. 3シナリオ提示フォーマット

商談時は以下のテーブルを埋めて顧客に提示してください。

| シナリオ | 前提 | 年間効果額 | 年間利用料 | ROI率 | 回収期間 |
| --- | --- | --- | --- | --- | --- |
| 保守的 | 改善率を半分に割り引く | | | | |
| 標準 | 仮説通り | | | | |
| 楽観的 | 改善率を1.5倍 | | | | |

保守的シナリオが黒字化しない場合は、提案の構成（対象業務の絞り込み、スタンダード移行タイミングの後ろ倒し等）を見直してください。

## 6. 注意書きと免責条項

- 本ワークシートの数値は顧客の実データで必ず再計算してください。商談メモや業界平均で埋めた数値は参考情報です。
- `ec-what-if` 予測モデルは現在精度再検証中のため、その出力を根拠にROIを約束することはできません。参考値として扱ってください。
- 本書は将来の業績、コスト削減額、売上増加額を保証するものではありません。
- ROI試算結果は意思決定の一材料であり、最終的な投資判断は顧客の責任において行われるものとします。
- 算出過程で使用した仮説・前提条件は必ず顧客と共有し、合意のうえで提案書に転記してください。
