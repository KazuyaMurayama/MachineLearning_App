# リードリスト フレームワーク

## 1. 背景と目的

現状、L3プロトタイプは完成しているにもかかわらず、リード獲得戦略が体系化されておらず、パイロット顧客も0社です。本書は、明日から動けるリード獲得フレームワークを提供することを目的としています。

目標は次のとおりです。

- **2週間以内**: 10社の見込リストを作成（名刺交換済みまたは紹介依頼済みの状態）
- **4週間以内**: 5社にパイロット提案完了（契約締結は稟議期間を考慮し6〜8週目標）

短期スプリントであるため、本書は「広げる」よりも「絞って動く」ことに重点を置いています。
