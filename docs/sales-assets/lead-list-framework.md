# リードリスト フレームワーク

## 1. 背景と目的

L3プロトタイプ完成済、パイロット顧客0社。明日から動けるフレームワークを提供。

- **2週間以内**: 10社の見込リスト作成
- **4週間以内**: 5社にパイロット提案完了（契約締結は稟議期間考慮で6〜8週目標）

## 2. 4チャネル戦略

### Ch1: 既存ネットワーク・紹介（60%🔸仮説）
- LinkedIn・名刺管理・個人メールから過去接点を洗い出し
- 1日5件目安で「6週間パイロットの5社を探している」文脈で連絡
- 歩留まり🔸仮説: 接触50件→面談15件→合意3件

### Ch2: 業界団体・勉強会（15%🔸仮説）
- 士業向け研究会・青色申告会等に登録、登壇機会優先
- 歩留まり🔸仮説: 接触20件→面談6件→合意1件

### Ch3: オンライン発信（15%🔸仮説）
- note/X/LinkedInで週2本、「学び9:告知1」比率
- 反応ある人にDMで30分対話を打診
- 歩留まり🔸仮説: DM返信10件→面談4件→合意1件

### Ch4: 限定的アウトバウンド（10%🔸仮説）
- ICP合致企業を公開情報から最20社、お問合せフォームor公式SNSに1通限り
- コールドコール禁止、追撃は1回のみ
- 歩留まり🔸仮説: 20件→面談2件→合意0〜1件

## 3. ICP定義

### 3.1 士業ICP
- **規模**: 職員5〜30名の税理士・会計士事務所
- **地域**: 全国（初期は対面可能地域優先）
- **ITリテラシー**: クラウド会計導入済 or チャットツール運用経験あり
- **痛みの兆候**: 顧問先離反の話題、経営相談拡大意向、所員定着課題
- **除外**: 零細(1-2名)、紙/FAX主体、類似AIに大型投資済

### 3.2 EC ICP
- **月商**: 5,000万円〜10億円
- **商材**: 自社企画商品、消耗型サブスク
- **プラットフォーム**: 自社EC+モール併用
- **痛みの兆候**: CAC/ROASが経営会議主要議題、代理店不満、データ見たいが時間ない
- **除外**: D2C立上げ直後、数万SKU超、フルオーダーBIコンサル契約中

## 4. リード評価スコア（簡易BANT）

合計12点満点。**8点以上**を優先リード。

- **B(Budget)**: 0なし/1仮検討/2裁量可能/3予算枚あり
- **A(Authority)**: 0現場のみ/1影響力/2決裁者パス可/3決裁者本人
- **N(Need)**: 0不明/1認識/2言語化済/3解決宣言
- **T(Timing)**: 0未定/1半年以上/2·3か月以内/3即時

毎週金曜更新。3週連続不変はステータス見直し。

## 5. リード台帳テンプレート（CSV列）

lead_id / 会社名 / 業種(士業|EC) / 規模 / 接点経路 / 決裁者 / ヒアリング済(Y|N) / BANTスコア(0-12) / 次アクション / 次アクション期日 / ステータス(未接触|接触済|面談設定|面談実施|提案中|契約|失注|保留) / 最終接触日 / 担当 / メモ

## 6. 初動30日ロードマップ

### Week1 (04-13〜04-19): 準備
- 月: ICP共有、台帳初期化
- 火〜水: 接点棚卸し→台帳登録(目標50件)
- 木: 優先10件選定
- 金: 優先10件に連絡開始

### Week2 (04-20〜04-26): 面談設定
- 返信対応、二次紹介依頼
- 業界団体リストアップ、オンライン発信開始
- **KPI: 見込リスト10社達成**

### Week3 (04-27〜05-03): ヒアリング
- `customer-pain-research.md` の15問テンプレで実施
- **KPI: 5社ヒアリング完了、う2社提案フェーズ移行**

### Week4 (05-04〜05-10): 提案・クロージング
- 提案書提出、稟議用サマリ同封
- **KPI: 5社提案完了**（契約締結は稟議期間後のWeek6-8目標）

## 7. アンチパターン

1. 一斉メール送信禁止
2. 個人情報の外部収集禁止
3. 定価の20%超の値引きはエスカレーション
4. 実績の誇張禁止（パイロット0社。「最初の5社」優位性として訠実に）
5. `ec-what-if` R²の断言禁止（`objection-handling.md` D-3参照）
6. 実名企業の内部メモ混在禁止
7. ヒアリング中の売り込み禁止（最初の20分は提案しない）
8. `customer-pain-research.md` の仮説数値を出典なしで営業資料に引用禁止
