# 営業アセット (Sales Assets)

L3パイロット顧客獲得のための営業資料を格納するディレクトリ。

## 含まれるファイル

| ファイル | 用途 |
|---------|------|
| `pilot-proposal-template.md` | パイロット運用提案書（士業版・EC版） |
| `demo-script-15min.md` | 15分デモのスクリプト（次フェーズ作成） |
| `roi-calculator.md` | ROI試算シート（次フェーズ作成） |
| `pricing-public.md` | 顧客向け料金表（次フェーズ作成） |

## 使い方

1. 商談相手の業種を確認（士業 or EC）
2. `pilot-proposal-template.md` の該当セクションをコピー
3. プレースホルダー（`{{顧客名}}` 等）を置換
4. PDF化して送付
