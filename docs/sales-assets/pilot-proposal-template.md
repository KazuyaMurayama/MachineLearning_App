# 【士業版】事務所AI経営パートナー パイロット運用提案

## 1. 課題認識

{{顧客名}} 様のような{{業種}}事務所が直面しやすい3つの典型的課題（いずれも弊社仮説、実態はヒアリング段階で検証いたします）:

1. **顧問先離反の予兆を掴みにくい** — 🔸仮説（要検証）: 1件失うと年間¥30〜120万円の損失レンジに及ぶ可能性
2. **月次経営会議の資料作成に時間を取られる** — 試算表→分析→資料化に毎月30分超
3. **採算の見えにくい顧問先が一定数存在する** — 全顧問先に同じ対応をしており、収益性に差が出ている可能性

## 2. 提供する解決策（L3パイロット・3アプリ）

### ① 事務所ダッシュボード
- 顧問先の離反スコア・入金状況・クロスセル機会を俯瞰
- 月次経営会議資料を短時間でMarkdown出力
- 担当者別・顧問先別の指標ドリルダウン

### ② LTV予測
- LightGBM + SHAP による顧問先の推定LTV（参考値。パイロット期間中に貴所データで再学習）
- VIP／成長／要注意 の自動クラスタ分類
- 改善アクションの試算（仮説ベース）

### ③ AIブリーフィング
- 離反リスクTOP5・入金遅延対応・翌月アクション3件を週次で自動生成
- 月次経営会議の資料を短時間で作成
- Markdown形式でダウンロード→印刷可能

## 3. パイロット運用の進め方（6週間）

| Week | 内容 | アウトプット |
|:----:|------|------------|
| 1 | キックオフ + 顧問先データのCSVアップロード・品質確認 | 初期分析レポート |
| 2 | 離反リスクTOP5顧問先の特定 + 対応策の議論 | アクションプラン |
| 3 | 実運用開始（週次スコア更新） | 週次サマリー |
| 4 | 中間レビュー + 機能調整 | フィードバック |
| 5 | LTV予測の精度レビュー + 現場適用状況確認 | 検証レポート |
| 6 | 最終レビュー + 本契約判断 | 効果測定報告 |

# 【EC事業者版】EC AIマーケティングパートナー パイロット運用提案

## 1. 課題認識

{{顧客名}} 様のような{{業種}}EC事業者が直面しやすい3つの典型的課題（いずれも弊社仮説、実態はヒアリング段階で検証します）:

1. **CAC高騰・ROAS悪化** — 🔸仮説（要検証）: チャネル別の最適化が後手に回りやすい
2. **休眠顧客の再エンゲージが難しい** — 🔸仮説: 既存顧客の一定割合が休眠化しているがアクション設計ができていない
3. **「データはあるが打ち手が決まらない」** — GA4等で「見る」ことはできても、「次の一手」が決まらない

## 2. 提供する解決策（L3パイロット・3アプリ）

### ① EC経営ダッシュボード【主役】
- 売上・広告・顧客・在庫を1画面で俯瞰
- RFMセグメント × チャネルROI ヒートマップ
- 離脱リスク・在庫切迫アラート
- 月次経営会議の論点を自動抽出

### ② AIブリーフィング
- チャネル別ROIレビュー
- 離脱リスク顧客TOP20 + 推奨アクション
- 翌月の重点アクション3件を自動生成

### ③ What-Ifシミュレーター【補助ツール・精度再検証中】

> **重要**: 本アプリの予測モデル `ec-what-if` は現在 R²=-0.16 と精度再検証中です。数値予測は確定値ではなく **参考値扱い** とし、パイロット期間中に貴社データで再学習します。主たる価値は「どの変数がどの指標に波及するか」の構造的示唆にあります。数値の確定的な提示は行いません。

- LightGBM + SHAP による感度分析（構造示唆）
- スライダーでのシナリオ比較
- 本アプリは補助ツールとして位置付けており、主役は ① EC経営ダッシュボードです

- 全機能はブラウザで動作（インストール不要）。運用は弊社の **ホステッド環境** で提供します
