# 【士業版】事務所経営AIパートナー パイロット運用提案

## 1. 課題認識

{{顧客名}} 様のような{{業種}}事務所が直面する3つの典型的課題:

1. **顧問先離反の予兆を掴めない**: 解約は突然訪れる。1社失うと年間¥30〜120万の損失
2. **月次経営会議の資料作成に時間を取られる**: 試算表→分析→資料化に毎月30分超
3. **不採算顧問先を特定できていない**: 全顧問先に同じ対応をしており、収益性に差がある

## 2. 提供する解決策（3つのプロトタイプ）

### ① 事務所経営ダッシュボード
- 8アプリのデータを1画面に集約
- 顧問先の離反スコア・入金状況・クロスセル機会を俯瞰
- 月次コンサル資料を5分でCSV出力

### ② 顧問先LTV予測+不採算フラグ
- LightGBM + SHAP で5年間のLTVを予測
- VIP/成長/不採算 の自動クラスタ分類
- 改善アクション3案をスライダー試算

### ③ 月次AIブリーフィングレポート
- 離反リスクTOP5・入金遅延対応・翌月アクション3件を自動生成
- 月次経営会議の資料を5分で作成
- Markdown形式でダウンロード→印刷可能
