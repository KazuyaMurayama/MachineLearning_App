## WTP推計ロジック

```
月額WTP = 課題の月間コスト影響額 × ソリューション寄与率 × 支払意欲係数(0.2)

- ソリューション寄与率: フィット★に応じて ★1=10% / ★2=25% / ★3=40% / ★4=60% / ★5=80%
- 支払意欲係数: 0.2（顧客は効果の20%まで支払う経験則）
- 全数値は仮説。パイロット検証前。
```
