# 06: 成果物の報告ルール

## ハイパーリンク必須（厳守）

成果物ファイルは **GitHub 上のリンク** をハイパーリンクで報告すること。

### フォーマット

```markdown
[ファイル名](https://github.com/KazuyaMurayama/MachineLearning_App/blob/main/パス/ファイル名)
```

### ルール

1. **ワンクリックで直接開けるリンク形式** にすること。
2. ブランチは原則 `main` を指定。
3. 複数ファイルの報告時はテーブル形式で一覧化。

### 例

| ファイル | リンク |
|---------|------|
| tasks.md | [tasks.md](https://github.com/KazuyaMurayama/MachineLearning_App/blob/main/tasks.md) |
| CLAUDE.md | [CLAUDE.md](https://github.com/KazuyaMurayama/MachineLearning_App/blob/main/CLAUDE.md) |

## 出力フォーマット

- md ファイルを優先。
- Google スプレッドシート・Word・PPT はユーザー指示があれば使用。
