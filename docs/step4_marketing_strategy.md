## 1. STP分析

### 1-1. Segmentation（市場細分化）

#### セグメンテーション変数

| 変数 | 分類基準 | セグメント |
|------|---------|-----------|

| **企業規模** | 従業員数 | 小規模（10〜49名） / 中規模（50〜299名） |
| **IT成熟度** | DX推進状況 | レベル1: Excel中心 / レベル2: 基幹システム導入済 / レベル3: データ活用検討中 |

### 1-2. Targeting（ターゲット選定）

**プライマリターゲット**: 従業員20〜150名の金属加工・樹脂成形メーカー（見積もりがベテラン依存）

| 属性 | 詳細 |
|------|------|
| 業種 | 金属加工（板金・切削・プレス）、樹脂成形（射出成形） |
| 従業員数 | 20〜150名 |
| 売上規模 | 3億〜30億円 |
| 見積もり件数 | 月50〜300件 |
| 主な課題 | ベテラン依存の見積もり、精度のバラつき、後継者不在 |
| IT環境 | Excel＋基幹システム。データはあるが活用できていない |
| 推定ターゲット企業数 | 全国約15,000〜20,000社 |

**セカンダリターゲット**: 同規模帯の製造業で歩留まり改善・リードタイム短縮に課題を持つ企業

### 1-3. Positioning（ポジショニング）

**ポジショニングステートメント**:

> 見積もりのベテラン依存に悩む中小製造業にとって、本アプリは「いつものExcel/CSVデータをアップロードするだけで、AIが見積もり金額を予測し、その根拠まで可視化する」唯一のノーコードツールです。

#### ポジショニングマップ

```
            高機能・カスタマイズ性
                    ↑
                    │
    DataRobot ●     │     ● 大手SIerの
    (高価格)        │       カスタム開発
                    │
    ────────────────┼────────────────→ 製造業特化
    汎用BI           │     ★ 本アプリ
    (Tableau等)     │     (ノーコード×
                    │      SHAP解釈)
                    │
                    │
            シンプル・即時利用
```

**差別化の3本柱**:

| # | 差別化要素 | 競合との違い |
|---|----------|------------|
| 1 | **ノーコード・即時利用** | DataRobot等は高額・複雑。本アプリはCSVアップロード→5分で予測 |
| 2 | **SHAP解釈可視化** | 「なぜその金額か」を可視化。ブラックボックスではない |
| 3 | **製造業特化の価格帯** | 月額3〜8万円。中小製造業のIT予算に合致 |

## 3. 4P/4C分析

### 3-1. 4P（企業視点）

#### Product（製品）

| 要素 | 内容 |
|------|------|
| **コア価値** | CSV/Excelデータで即座に回帰予測＋SHAP解釈 |
| **基本機能** | データアップロード、変数選択、LightGBM学習、予測実行、SHAP可視化、CSV出力 |
| **付加機能（フェーズ1）** | 見積もりテンプレート、操作チュートリアル |
| **付加機能（フェーズ2以降）** | API連携、バッチ予測、モデル管理、チーム機能 |
| **品質基準** | 予測精度R²>0.8を目安、応答時間3秒以内 |

#### Price（価格）

| プラン | 月額 | 年額 | 対象 | 含まれる機能 |
|--------|------|------|------|------------|
| **ライト** | 30,000円 | 324,000円（10%割引） | 個人事業・小規模 | 月50回予測、データ100MB |
| **スタンダード** | 50,000円 | 540,000円（10%割引） | 中小企業（推奨） | 月200回予測、データ1GB、SHAP解析 |
| **プロ** | 80,000円 | 864,000円（10%割引） | 中堅企業 | 無制限予測、データ5GB、優先サポート |
| **無料トライアル** | 0円 | — | 全対象 | 14日間、全機能利用可 |

**価格設定根拠**:
- 見積もり1件あたり30分→10分に短縮 = 月20件で月10時間削減
- 時給換算3,000円 × 10時間 = 月30,000円の人件費削減（ライトプランでペイ）
- 精度向上による受注率改善効果（年間数百万円のインパクト）
