## 1. 事業計画のEC向けサービス構造（L1/L2/L3）— v2.1整合

事業計画（§11C + §12B）では、明示的な "L1/L2/L3" ラベルは使用していないが、実質的な3層構造が定義されている。

### 価格層の対応表

| 層 | パッケージ名 | 月額 | 含まれるアプリ数 | 主な機能 | 状態 |
|----|------------|------|----------------|---------|------|
| **L1** | PoC階段 | **¥60,000/月（3ヶ月固定）** | 1本 | ML予測1本+月1MTG+IT補助金併用 | パイロット運用準備中 |
| **L2** | 人+AIバンドル | **¥170,000/月** | 4本+月次MTG | ML予測4本+SHAP+月1施策MTG+月次レポート+商品登録/SNS自動化 | パイロット検証段階 |
| **L3** | 戦略伴走 | **¥250,000/月** | 12本+伴走 | L2全機能+月2MTG+カスタムML+四半期戦略レビュー | **プロトタイプ・未パイロット** |

**価格帯（空白地帯）**: ¥60,000-250,000「人+AI＋日本特化（薬機法/景表法/季節性）＋IT補助金対応」帯（自社定義、phase2文書と統一）

### 類似競合の価格帯（v2.1拡充）

| 競合カテゴリ | 代表ツール | 月額 | 脅威度 |
|-------------|----------|------|--------|
| **Shopify Magic / Sidekick** | Shopify公式AI | **無料**（Shopify同梱） | **高** |
| BIツール | Looker Studio / Tableau / PowerBI | 無料-20万円 | 中 |
| **Klaviyo（CRM内蔵AI）** | Klaviyo | $150-数十万円 | **高** |
| **汎用AI** | GPT-5 / Claude Opus 4.x + Code Interpreter | ¥3,000/月 | **高** |
| 需要予測SaaS | UMWELT / FOREMAST | 5千-50万円 | 低 |
| CRM/LTV | b→dash / aimstar | 3-30万円 | 中 |
| 国産ECコンサル人力 | StoreHero / コマースメディア等 | 30-100万円 | 中 |
| 個別AI開発 | DataRobot / H2O.ai | 20-100万円 | 低 |
| **当社L1-L3** | EC横断分析+人+AI | **6-25万円** | — |

**差別化軸**: ① IT導入補助金対応（実質1/2負担）② 3ヶ月PoC階段（¥6万から始める）③ 人+AI伴走（純SaaSをやめる、汎用AI単独では代替不能な月1MTG）

**用語の戦略的撤退**:
- × 「逆SHAP What-Ifシミュレーター」（バズワード化リスク。SHAPは2020年からの標準手法、DataRobot等が同等保有）
- ○ **「ML予測+月次レビュー+IT補助金対応+日本特化」**

### L3キラー機能・競合差別化

- **ML予測+月次レビュー（顧客視点での意思決定支援）**: GA4・Lookerにはない機能。ec-demoのアクション提案をec-ad-roi/ec-dashboardへ横展開
- **横断分析（共通データ基盤）**: 「1回アップロードで全アプリ連携」でロックイン効果
- **月次自動レポート**: 月次解約率を1.2%以下に抑えるロックイン効果
- **最大の差別化**: CSVアップロードだけでLightGBM + SHAP + 月次レビューまでワンストップ提供

## 5. 結論（v2.1整合）

### L3プロトタイプ完成・パイロット検証前

当初「現在の4 MVP は L1〜L2 の下限に位置し、L3到達には3つの急務がある」と特定したが、以下3本のL3キラー機能のプロトタイプを新規実装し、パイロット検証ステージへ移行。**L3戦略伴走パック（¥250,000/月）プロトタイプ水準に到達（パイロット検証前）**。

**v2.1事業計画と整合**: L3 ¥250,000/月パッケージの**販売開始は、パイロット顧客1社×6週間以上の実運用検証完了をもってのみ可能**。それ以前は「L3プロトタイプ・パイロット検証前」ステージにある。

**2026年5月時点で、当社EC アプリ群は技術的にL3水準に達しているが、商業稼働には至っていない。**

| 新規アプリ | ディレクトリ | 主要機能 | 品質スコア |
|---|---|---|:---:|
| EC経営ダッシュボード | `apps/ec-executive-dashboard/` | 4アプリデータ統合、RFMセグメント×チャネルROIヒートマップ、在庫・離脱リスクアラート | 95/100 |
| What-Ifシミュレーター | `apps/ec-what-if/` | LightGBM 3モデル（売上/ROAS/離脱率）+SHAP beeswarm+スライダー試算+3シナリオ比較 | 95/100 |
| 月次AIブリーフィング | `apps/ec-monthly-briefing/` | ルールベースエグゼクティブサマリー、離脱リスクTOP20+アクション、翌月施策3件、Markdown出力 | 95/100 |

> 品質スコア=技術的完成度。事業価値は1社×6週間以上のパイロット検証完了でのみ確定

**次フェーズ**: M1-M2でEC事業者3-5社へL1¥6万PoCアウトリーチ、M3-M4で1社契約獲得 → M5-M6でL2¥17万へアップセル試行 → M7-M9でL3¥25万 production-ready 化

### 当初3つの急務 → 解消状況

1. ✅ **逆SHAP What-Ifシミュレーター** → `ec-what-if` で実装（用語は「ML予測+What-Ifシミュレーション」に転換）
2. ✅ **共通データ基盤による横断分析** → `ec-executive-dashboard` で 4アプリのデータを1画面統合
3. ✅ **月次自動レポート+改善提案サマリー** → `ec-monthly-briefing` でルールベース文章生成+Markdownレポート出力を実装
4. ⚠️ **新規残課題**: ec-dashboard へのML予測+SHAP追加（L1完成のボトルネック）、Shopify公式機能との補完ポジション戦略の確定（M3まで）
