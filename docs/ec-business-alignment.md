## 1. 事業計画のEC向けサービス構造（L1/L2/L3）

事業計画（§11C + §12B）では、明示的な "L1/L2/L3" ラベルは使用していないが、実質的な3層構造が定義されている。

### 価格層の対応表

| 層 | パッケージ名 | 月額 | 含まれるアプリ数 | 主な機能 |
|----|------------|------|----------------|---------|
| **L1（スターター）** | スターターパック | ¥125,000 | ベース3本（売上予測 + 離脱予測 + 顧客セグメンテーション）| 基礎ML予測・SHAP分析 |
| **L2（グロース）** | グロースパック | ¥170,000 | 6本（L1 + LTV予測 + 在庫ABC + アソシエーション）| 横断分析・LTV最大化 |
| **L3（プレミアム）** | プレミアムパック | ¥250,000 | 12本 + 月次レビュー + カスタマイズ | 全機能 + 伴走支援 |

構造の詳細（§11C 改善版）:
- **ベース月額**: ¥80,000（コア3アプリ + 共通データ基盤）
- **追加アプリ**: ¥15,000/月/アプリ（段階アンロック）
- **データ活用顧問**: ¥100,000/月（オプション）
- **最上位到達時ARPU**: M24で平均¥230,000（標準シナリオ）、最大¥405,000（ベース+15アプリ+顧問）

### 顧客ペインと支払能力

- **ターゲット**: EC年商1億〜100億円の中堅EC事業者（国内5,000〜10,000社）
- **ペインの核心**: CAC高騰（コスト増）+ 離脱予測不能 + DS不在 + 複数SaaS乱立コスト
- **支払能力**: 月額8万〜20万円（EC事業者はROI志向が強く、効果実証後は支払意欲高）
- **競合比較**: 個別SaaS3本契約（13万〜110万円/月）vs 当社L2（17万円/月）→ 価格優位あり

### 類似競合の価格帯

| 競合カテゴリ | 代表ツール | 月額 |
|-------------|----------|------|
| BIツール | Looker, Tableau, Power BI | 5万〜20万円 |
| 需要予測SaaS | UMWELT, FOREMAST | 5千〜50万円 |
| CRM/LTV分析 | b→dash, aimstar | 3万〜30万円 |
| 汎用MLプラットフォーム | DataRobot, MatrixFlow | 20万〜100万円 |
| GA4 | Google | 無料〜（BigQuery連携で有料化） |
| **当社ポジション** | EC横断分析 | **8万〜25万円**（空白地帯） |

**差別化軸**: GA4/Lookerが「可視化」、DataRobotが「精度」に特化するのに対し、当社は「横断分析 + SHAP説明可能性 + 逆SHAP（アクション提案）」の組み合わせで空白地帯を占める。

## 2. 現在のEC MVP 4アプリの層別評価

### 2.1 `ec-demo` — 顧客離脱予測 + 需要予測

**対応層: L1** — L1要求充足率: 70%（逆SHAPは先行実装だが技術負債とデータ基盤が未対応）

### 2.2 `ec-dashboard` — 売上ダッシュボード

**対応層: L1（下位）** — L1要求充足率: 45%（可視化はできているが予測とSHAPが欠如）

### 2.3 `ec-rfm` — 顧客RFM分析

**対応層: L2** — L2要求充足率: 60%（RFM集計はできているがアクション提案と他アプリ連携が欠如）

### 2.4 `ec-ad-roi` — 広告ROI分析

**対応層: L2〜L3** — L2要求充足率: 55%（集計・可視化はL1相当だが、ML予測とSHAPでL2〜L3に届かない）

### L3キラー機能・競合差別化

- **逆SHAP What-Ifシミュレーター**: GA4・Lookerにはない機能。ec-demoのACTION_TEMPLATESをec-ad-roi/ec-dashboardへ横展開
- **横断分析（共通データ基盤）**: 「1回アップロードで全アプリ連携」でロックイン効果
- **月次自動レポート**: 月次解約率を1.2%以下に抑えるロックイン効果
- **最大の差別化**: CSVアップロードだけでLightGBM + SHAP + 逆SHAPまでワンストップ提供

## 5. 結論（2026-04-08 更新）

### L3プロトタイプ完成・パイロット検証前

当初「現在の4 MVP は L1〜L2 の下限に位置し、L3到達には3つの急務がある」と特定したが、以下3本のL3キラー機能のプロトタイプを新規実装し、パイロット検証ステージへ移行。**L3プレミアムパック（¥250,000〜¥405,000/月）プロトタイプ水準に到達（パイロット検証前）**。

| 新規アプリ | ディレクトリ | 主要機能 | 品質スコア |
|---|---|---|:---:|
| EC経営ダッシュボード | `apps/ec-executive-dashboard/` | 4アプリデータ統合、RFMセグメント×チャネルROIヒートマップ、在庫・離脱リスクアラート | 95/100 |
| What-Ifシミュレーター | `apps/ec-what-if/` | LightGBM 3モデル（売上/ROAS/離脱率）+SHAP beeswarm+スライダー試算+3シナリオ比較 | 94/100 |
| 月次AIブリーフィング | `apps/ec-monthly-briefing/` | ルールベースエグゼクティブサマリー、離脱リスクTOP20+アクション、翌月施策3件、Markdown出力 | 95/100 |

### 当初3つの急務 → 解消状況

1. ✅ **逆SHAP What-Ifシミュレーター実装** → `ec-what-if` で LightGBM+SHAP+スライダー試算を完成
2. ✅ **共通データ基盤構築による横断分析** → `ec-executive-dashboard` で 4アプリのデータを1画面統合
3. ✅ **月次自動レポート+改善提案サマリー** → `ec-monthly-briefing` でルールベース文章生成+Markdownレポート出力を実装

残課題: **ec-dashboard へのML予測+SHAP追加**（L1スターター完成のボトルネック）は未着手

### 次フェーズ（L3パイロット顧客獲得）

- EC年商1億〜100億円の中堅EC事業者1社でのパイロット運用
- 月額25万円〜40.5万円（ベース+オプション）の価格実証
- 競合比較: GA4/Looker（5-20万円）は「見る」ツール、DataRobot（20-100万円）は「精度」、当社は「次のアクションを教える」ツール
- 目標: L3 1社 + L2 2〜3社 = 月50〜80万MRR

### 技術的な学び

当初「ec-demo の ACTION_TEMPLATES が事業計画の逆SHAPに最も近い」と特定した設計思想を、`ec-what-if` にて LightGBM 3モデル並列学習 + SHAP TreeExplainer + スライダーUIという形で横展開することに成功。これにより「見る→動く」ツールへの進化を実現。

## ⚠️ 注記: L3水準到達の定義

「L3水準到達」とは、技術的なプロトタイプが完成した段階を指す。月額20万円（士業）/25万円（EC）の最終充足には、**パイロット顧客1社による6週間以上の実運用検証** をもって確定する。それまでは「プロトタイプ完成・パイロット検証前」のステージにある。

- パイロット検証完了 → 「L3 production-ready」と表記
- 売上実績 1社以上獲得 → 「L3 商業稼働」と表記
- 現在は **L3プロトタイプ完成** ステージ
