## 3. 士業セグメント（80点・1位）

### 3.1 3エージェント議論サマリー

#### 議論①: 月10hで税理士にリーチできるか？

- **楽観**: note/X/LinkedInで「税理士事務所×データ分析」記事を月2本。税理士はLinkedInで積極発信しており、コメント→DM→Zoomデモの導線が作れる。6ヶ月で5件のリード。
- **悲観**: 税理士の情報収集はTKC・船井総研・会計人コースなどの専門チャネルに偏る。個人ブログが目に留まる確率は極めて低い。noteの記事がバズっても「面白い」で終わり、問い合わせには繋がらない。
- **リアリスト**: LinkedInは有効チャネルだが、税理士のLinkedIn利用率はまだ低い。note＋Xで「顧問先分析」「解約予測」の具体的事例を示し、freee/MFのパートナーコミュニティに投稿する方が接点を作りやすい。

#### 議論②: 単価¥50,000〜¥100,000は取れるか？

- **楽観**: 船井総研のコンサルが月¥300,000。その1/3以下でデータドリブンな分析ができるなら¥100,000は安い。5事務所の顧問先分析で月¥500,000。
- **悲観**: 税理士事務所の平均IT予算は月¥3〜10万円。既にTKC・JDL・freeeに払っている中で追加¥50,000は高い。実績ゼロの一人法人には¥30,000が限界。
- **リアリスト**: 初期は¥30,000で実績を作り、効果実証後に¥50,000〜¥80,000にアップセル。船井総研のコンサルと比較した「コスパ訴求」が刺さる。

#### 議論③: 解約率と月10hでのサポート

- **楽観**: 税理士の顧問契約は長期型。同様にツール契約も年単位で継続。解約率は月2%程度。
- **悲観**: 税理士事務所は繁忙期（1〜3月、5月）にツールを見直す余裕がなく、閑散期（6〜8月）に「これ使ってたっけ」と解約される。月3〜4%。
- **リアリスト**: 顧問先の分析レポートを月次自動配信すれば、解約率は月2〜3%に抑えられる。月10hでは7〜8社が上限。

## 4. 小売/ECセグメント（75点・2位）

### 4.1 3エージェント議論サマリー

#### 議論①: 月10hでEC事業者にブログ何本書ける？

- **楽観**: Claude Code活用で月3〜4本。EC事業者は「売上予測」「LTV」「離脱防止」で能動検索。SEO記事がストック資産に。Shopifyアプリストアに載せれば露出も取れる。
- **悲観**: 既存顧客サポート・バグ修正にも時間を取られ、現実はブログ月1本が限界。EC領域のSEOはShopify公式ブログやECのミカタ等が上位独占。個人ブログが検索上位に来るまで12ヶ月以上。
- **リアリスト**: Claude Code効率化を考慮して月1.5〜2本が現実的。SEO成果は6〜9ヶ月後。初期はShopifyコミュニティ投稿、X/Twitter、EC系Slackを併用すべき。

#### 議論②: 無料ツール（GA4等）との差別化は月10hで可能か？

- **楽観**: GA4は汎用ツール、EC特化の売上予測やLTV分析はできない。「GA4では分からない来月の売上を予測する」という切り口で差別化可能。
- **悲観**: EC事業者がそれに月¥50,000〜¥100,000払うかが問題。年商1億規模は少人数運営でML予測を運用する余裕がない。年商10億以上は大手コンサルを使う。ターゲットの年商帯が中途半端。
- **リアリスト**: 年商3〜30億のEC事業者がスイートスポット。GA4との違いを具体数字で示す必要（例:「LTV予測で広告ROASが20%改善」等）。最初の1〜2社は無料〜格安で実績を作るフェーズが不可避。

#### 議論③: Shopifyアプリストア掲載の現実性

- **楽観**: Shopifyコミュニティは活発。有用な投稿をすれば3ヶ月で認知。アプリストア掲載でさらに加速。
- **悲観**: Shopifyアプリストアの審査は厳格で掲載まで2〜3ヶ月。掲載後もレビューが少ないアプリは埋もれる。月10hではコミュニティ投稿頻度が足りない。
- **リアリスト**: アプリストア掲載準備に2〜4ヶ月、認知獲得に追加3〜6ヶ月。合計6〜10ヶ月。初期はアプリストア以外（直販・紹介）でのトラクション獲得が現実的。

**リアリスト（最終結論）**: 「**士業一本集中が最適解**。理由:

1. **§12Gスコア1位（80点）**: リーチ・プロモ・契約・開発・収益の全軸でバランスが良い
2. **LTV最高（¥300万）**: 解約率の低さ×単価の高さで、少数顧客でも安定収益
3. **紹介文化**: 月10hでは積極営業が困難だが、紹介ベースなら工数をかけずに顧客獲得可能
4. **競合環境**: 船井総研は月¥30万。¥6〜8万の価格帯に直接競合が少ない
5. **初期キャッシュフロー**: 5ヶ月目に初売上の可能性。歯科の8ヶ月と比較して資金繰りリスクが低い

**目標設定**:
- **保守的目標**: 24ヶ月後 ARR ¥3,000,000（悲観〜標準の間）
- **標準目標**: 24ヶ月後 ARR ¥5,760,000
- **ストレッチ目標**: 24ヶ月後 ARR ¥8,000,000（月10h→15hへの微増を前提）
