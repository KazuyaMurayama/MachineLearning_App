## 1. エグゼクティブサマリー

**小売/ECは、食品製造業に次ぐ第2優先セグメントであり、特にデータ量の豊富さとML活用親和性の高さが際立つ。**

| 項目 | 評価 |
|------|------|
| 市場規模 | 国内BtoC-EC市場 約26兆円（2024年）、小売業全体 約150兆円 |
| ターゲット企業数 | EC年商1億〜100億円の中堅EC事業者 約5,000〜10,000社 |
| ML/DS関連ペインの深刻度 | ★★★★★（需要予測・LTV・パーソナライズ・在庫最適化が同時発生） |
| IT投資意欲 | ★★★★★（EC事業者はデータドリブン志向が高い） |
| 当社ソリューションとの適合度 | ★★★★★（購買予測・顧客分析・離脱予測がコアニーズ） |
| 競合強度 | ★★★★☆（レコメンド系は激戦だが、ML分析ツールは差別化可能） |
| 想定ARPU | 月額 8万〜20万円（ベース＋アプリアンロック） |
| 参入優先度 | **#2（高優先）** |

**選定理由のサマリー**:
- EC事業者は購買データ・行動データが豊富で、ML導入の前提条件が整っている
- 顧客獲得コスト（CAC）高騰により、LTV最大化・離脱防止の経済インパクトが大きい
- 当社のLightGBM回帰/分類モデルが売上予測・離脱予測・LTV分析に直接活用可能
- SHAP解析による「なぜこの顧客が離脱するのか」の説明が強い差別化要素
- 食品製造業との共通技術基盤でアプリ開発効率が高い（横展開コスト低）

## 2. 小売/EC業界の概況

### 2.1 市場規模と構造

| 指標 | 数値 | 出典 |
|------|------|------|
| 国内BtoC-EC市場規模 | 約26兆円（2024年） | 経済産業省 |
| EC化率 | 約10%（主要国では低水準、成長余地大） | 経済産業省 |
| 世界EC市場規模予測（2026年） | 7.9兆ドル（約930兆円） | eMarketer |
| AI in Retail市場（グローバル） | USD 142億（2025年）→ USD 961億（2030年予測） | 各種リサーチ |
| AI in Retail CAGR | 46.54%（2025-2030） | StartUs Insights |
| 日本の小売市場 | USD 1.8兆（2025年） | Futurism |
| 小売業の非正規社員不足率 | 48.9% | 労働政策研究・研修機構（2024年） |

### 2.2 業界の構造的特徴

1. **EC化率の伸びしろ**: 日本のEC化率は約10%で世界平均を下回り、今後の成長ポテンシャルが大きい
2. **D2Cモデルの台頭**: メーカー直販で1st Party Dataを保有、CRM/分析ニーズが急増
3. **OMO（Online Merges with Offline）**: 実店舗とECの顧客データ統合が競争力の源泉に
4. **リテールメディア**: ECサイトを「広告媒体」化する動き（大手小売の第2の収益源）
5. **対話型コマース**: ChatGPTショッピング機能等、AI主導の購買体験が2026年のメガトレンド
6. **物流2024年問題の残存**: ドライバー不足・コスト高が継続的な経営課題
7. **CAC高騰**: デジタル広告単価の上昇で、新規獲得より既存顧客のLTV最大化にシフト

### 2.3 DX/AI化の現状

- ECサイトの約48%がAI試験段階を終了、29%が本格導入済み（Salesforce調査）
- AIレコメンド導入サイトで**17%の売上アップ**事例も
- 「2025年デジタルの崖」問題 — DX未対応企業は年間12兆円の損失リスク（経産省）
- 中小EC事業者では依然としてExcelベースの在庫管理・需要予測が主流
- データサイエンス人材の不足が導入障壁

## 3. ターゲット企業群の定義

### 3.1 プライマリターゲット（最優先）

**EC年商1億〜100億円の中堅EC事業者（推定 約5,000〜10,000社）**

| 条件 | 詳細 |
|------|------|
| EC年商 | 1億〜100億円 |
| 従業員数 | 10〜200名 |
| 事業形態 | 自社ECサイト運営（+モール併用） |
| IT部門 | 少人数（0〜5名）、専任データ分析者はいない |
| IT/マーケ予算 | 年間 300万〜3,000万円 |
| DXステージ | GA4等の基本ツールは導入済み、ML活用はこれから |

**代表的なサブセグメント**:

| サブセグメント | 特徴 | ペインの特徴 |
|---------------|------|-------------|
| D2Cブランド | 1st Party Data豊富、LTV重視 | 離脱予測、CRM最適化、LTV分析 |
| アパレルEC | 季節・トレンド変動大、返品率高 | 需要予測、在庫最適化、サイズレコメンド |
| 食品EC/サブスク | 定期購入、賞味期限管理 | 解約予測、需要予測、配送最適化 |
| 雑貨・インテリアEC | SKU多数、ロングテール | 在庫最適化、レコメンド、ABC分析 |
| BtoB EC | 取引先別価格、受発注最適化 | 需要予測、顧客セグメンテーション |
| 多店舗EC（楽天+自社+Amazon） | データ分散、在庫一元管理 | データ統合、在庫最適化、売上予測 |

### 3.2 セカンダリターゲット

- **中小小売チェーン**（5〜50店舗）: 店舗別需要予測、人員最適配置
- **ECモール出店者**（楽天・Yahoo!ショッピング）: 売上予測、広告ROI分析
- **EC支援会社/代理店**: 顧客への付加価値サービスとして当社ツールを提案

### 3.3 食品製造業セグメントとの相乗効果

- 食品メーカーのD2C展開 → 食品製造 + EC の両面でサービス提供可能
- 需要予測の技術基盤は共通（LightGBM回帰）→ 開発コスト低
- 食品EC（サブスク型）は食品製造業セグメントのクロスセル先

## 4. ペイン・課題の詳細分析

### 4.1 需要予測・在庫最適化

**深刻度: ★★★★★ | 経済インパクト: 過剰在庫20〜30%削減、欠品24%減少**

**現状の課題**:
- EC事業者のSKU数は増加の一途（多品種・小ロット化）
- 季節変動、トレンド変動、プロモーション効果の予測が困難
- 過剰在庫 → セール値引き・廃棄 → 利益率低下の悪循環
- 欠品 → 機会損失 + 顧客満足度低下
- 中小EC事業者はExcel or 経験ベースの発注が主流

**経済的インパクト（EC年商10億円企業の場合）**:
- 過剰在庫による値引き損: 年間 3,000万〜5,000万円
- 欠品による機会損失: 年間 1,000万〜2,000万円
- AI需要予測で過剰在庫20〜30%削減 → **年間 600万〜1,500万円の効果**
- 在庫回転率改善 → キャッシュフロー改善

### 4.2 顧客離脱・LTV最大化

**深刻度: ★★★★★ | 経済インパクト: LTV 15〜30%向上**

**現状の課題**:
- 新規顧客獲得コスト（CAC）が年々上昇（デジタル広告CPCの高騰）
- 既存顧客の維持・LTV向上が収益性の鍵だが、体系的に取り組めていない
- 「どの顧客が離脱しそうか」を予測できず、事後対応に追われる
- サブスクリプション型ECでは解約率（チャーンレート）が収益を左右
- RFM分析は行っているが、高度な予測分析は未導入

**経済的インパクト（EC年商10億円・定期購入型）**:
- 月次解約率を5%→4%に改善するだけで、年間売上 約1.2億円の差
- 離脱予測で上位20%のリスク顧客に施策集中 → チャーン率10〜20%改善
- LTV 15〜30%向上 → **年間利益 1,000万〜3,000万円の改善**

### 4.3 パーソナライゼーション・レコメンド

**深刻度: ★★★★☆ | 経済インパクト: CVR 10〜35%向上**

**現状の課題**:
- 顧客一人ひとりに最適な商品提案ができていない
- 「この商品を買った人はこの商品も買っています」程度の単純レコメンドが主流
- 行動データ（閲覧・カート・購買・回遊）の統合分析ができていない
- リアルタイムのパーソナライズ対応は技術的ハードルが高い
- ECと実店舗の顧客行動データが統合されていない（OMO課題）

**経済的インパクト**:
- AIレコメンド導入でEC売上 17%向上（Salesforce調査）
- Amazonでは売上の35%がAIレコメンド経由（McKinsey）
- まんだらけ: AIレコメンド導入で**月額2,000万円以上の売上増**

### 4.4 顧客獲得コスト（CAC）高騰

**深刻度: ★★★★☆ | 経済インパクト: 広告ROI 20〜40%改善**

**現状の課題**:
- Google/Meta広告のCPC（クリック単価）が年5〜15%上昇
- 新規獲得の限界効用が逓減（広告費をかけても効率が悪化）
- 広告チャネル別のROI/ROAS分析が不十分
- 顧客獲得チャネルとLTVの相関分析ができていない
- CACとLTVのバランスを定量的に管理できていない

**データ分析による改善**:
- チャネル別LTV分析 → 高LTV顧客を獲得しやすいチャネルに予算集中
- 類似オーディエンス（Lookalike）の精度向上 → CPC最適化
- 広告投資のROI予測モデル → 無駄な広告費の削減

### 4.5 オムニチャネル（OMO）データ統合

**深刻度: ★★★☆☆ | 経済インパクト: 統合顧客理解による売上5〜15%向上**

**現状の課題**:
- 実店舗（POS）とEC（行動ログ）のデータが別システムで管理
- 同一顧客のオフライン/オンライン行動を紐付けできない
- 楽天・Amazon・自社ECの販売データが分断
- 統合データ基盤の構築コストが高い（数千万円〜）
- 統合後のデータ活用方法が不明確

**当社にとっての意味**:
- データ統合そのものは大手SIerの領域（当社の直接競合ではない）
- **統合後のデータ分析・予測モデル構築が当社の価値提供領域**
- CSVエクスポートレベルでもML分析は可能（完全統合を待たずに開始）

### 4.6 物流・配送最適化

**深刻度: ★★★☆☆ | 経済インパクト: 物流コスト10〜20%削減**

**現状の課題**:
- 物流2024年問題（ドライバー時間外労働規制）の影響が継続
- 配送コストの上昇がEC事業者の利益を圧迫
- 配送日時の需要予測（いつどこにどれだけ配達が集中するか）が不十分
- 返品・交換の処理コストが高い（特にアパレル）
- ラストワンマイルの効率化が課題

**当社アプリでの対応範囲**:
- 配送需要予測（回帰モデル） → 配車計画の最適化に貢献
- 返品予測（分類モデル） → 返品率の高い商品/顧客の特定

## 6. 求められるソリューション

### 6.1 売上・需要予測AI

**ニーズの強さ: ★★★★★ | 当社対応可能度: ★★★★★**

| 要素 | 詳細 |
|------|------|
| 求められる機能 | 日次〜週次のSKU別売上予測、プロモーション効果予測 |
| データ入力 | 過去販売実績、季節指数、広告費、天候、イベント |
| 出力 | SKU別予測値、信頼区間、変動要因（SHAP） |
| 精度目標 | 予測誤差20〜40%改善（従来比） |
| 当社アプリ対応 | **LightGBM回帰予測アプリがそのまま活用可能** |

**顧客が求める要件**:
1. 「なぜ売れる/売れないのか」の要因可視化（SHAP）
2. プロモーション投入時のWhat-if分析
3. 新商品のコールドスタート問題への対応
4. CSVアップロードだけで使える簡便さ

### 6.2 顧客セグメンテーション

**ニーズの強さ: ★★★★☆ | 当社対応可能度: ★★★★★**

| 要素 | 詳細 |
|------|------|
