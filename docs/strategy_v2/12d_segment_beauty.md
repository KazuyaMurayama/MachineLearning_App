## 1. エグゼクティブサマリー

### 美容業界×ML予測アプリの機会

**市場規模**: AI in Beauty & Cosmetics市場は$4.38B（2025年）→ $5.3B（2026年）、CAGR 21.1%で急成長中。日本の美容室数は25.8万店（過去最高）で競争激化が進み、データ活用による差別化ニーズが急拡大している。

**セグメント評価スコア: 3.7 / 5.0**（Phase 2推奨）

| 評価項目 | スコア | 根拠 |
|---------|--------|------|
| 市場規模・成長性 | ★★★★☆ | 25.8万店、DX浸透率低く伸び代大 |
| ペインの深さ | ★★★★☆ | リピート率・離反が経営直結 |
| 技術適合性 | ★★★★☆ | LightGBM回帰が来店予測に直接適用可 |
| 競合状況 | ★★★☆☆ | 予約SaaSは飽和、ML分析は未開拓 |
| 導入障壁 | ★★★☆☆ | デジタルリテラシー格差が大きい |
| 単価ポテンシャル | ★★★★☆ | 多店舗チェーンは月額10万円以上可 |

**核心的インサイト**:
- 美容業界の最大ペインは**顧客離反**（新規集客コストは既存維持の5〜7倍）
- 既存SaaS（SALON BOARD, BEAUTY POS等）は予約管理中心で**予測分析機能が欠如**
- AI活用事例：ノーショー（無断キャンセル）予測で50.7%削減、売上予測精度95%
- 補助金活用で初期導入コストを1/2〜2/3に圧縮可能（IT導入補助金等）

**推奨戦略**: 多店舗展開サロンチェーン（5店舗以上）をメインターゲットに、離反予測AI + 売上予測をコアとしたスイート提供。Phase 1（食品製造・小売EC）で確立した基盤技術を水平展開し、美容特化のUI/UXでローカライズする。

## 3. ターゲット企業群の定義

### 3.1 セグメンテーション

| セグメント | 店舗数 | 年商規模 | 推定企業数 | 優先度 |
|-----------|--------|---------|-----------|--------|
| **Tier 1: 大手チェーン** | 50店舗以上 | 10億円以上 | 約50社 | ★★★★★ |
| **Tier 2: 中規模チェーン** | 5〜49店舗 | 1〜10億円 | 約500社 | ★★★★★ |
| **Tier 3: 小規模多店舗** | 2〜4店舗 | 3,000万〜1億円 | 約3,000社 | ★★★★☆ |
| **Tier 4: 個人サロン** | 1店舗 | 〜3,000万円 | 約20万店 | ★★☆☆☆ |

### 3.2 メインターゲット：Tier 1-2（多店舗チェーン）

**選定理由**:
- **データ量**: 複数店舗のPOS・予約データが蓄積されており、ML学習に十分
- **投資余力**: 月額10〜30万円のSaaS導入が可能な財務基盤
- **経営課題の顕在化**: 店舗間の業績差、スタッフ配置、在庫管理が複雑化
- **意思決定者**: 本部にデータ活用への理解がある経営層が存在
- **スケーラビリティ**: 1社契約で複数店舗分の収益

**代表的な企業例**:
- EARTH（220店舗）、AFLOAT（15店舗）、ALBUM（10店舗）
- Agu.（約900店舗、FC含む）、LIPPS（20店舗）
- 地方チェーン：各地域5〜30店舗規模

### 3.3 サブターゲット：Tier 3（小規模多店舗）

**狙い方**: Tier 1-2で成功事例を作った後、簡易版（ライト版）で展開
- 月額¥30,000〜¥50,000の低価格帯
- セルフサービス型のダッシュボード
- SALON BOARD/HOT PEPPER Beauty連携による自動データ取込

### 3.4 TAM/SAM/SOM

```
TAM（潜在市場）= Tier 1-3 × 平均月額
```

## 4. ペイン・課題の詳細分析

### 4.1 リピート率・顧客離反

**ペインの深さ: ★★★★★（最重要）**

| 指標 | 数値 | インパクト |
|------|------|-----------|
| 業界平均リピート率 | 60〜70% | 3割の顧客が毎年離反 |
| 新規獲得コスト vs 維持コスト | 5〜7倍 | 離反1人の損失 = 新規5人分のコスト |
| 離反理由の不可視性 | 80%以上 | 「なんとなく」離反の原因が特定不能 |
| 失客後の復帰率 | 5%以下 | 一度離反すると戻らない |

**現状の対応**:
- DMハガキ・LINE一斉配信（非パーソナライズ）
- スタイリストの記憶頼み
- 来店間隔が空いた顧客への手動フォロー（遅すぎる）

**ML解決の方向性**:
- 来店履歴・メニュー履歴・間隔パターンからの**離反リスクスコアリング**
- LightGBM回帰で「次回来店までの予測日数」を算出
- 閾値超過時に自動アラート → 先手のフォロー施策

### 4.2 集客コスト高騰

**ペインの深さ: ★★★★☆**

| チャネル | 月額コスト | CPO（1人あたり獲得コスト） |
|---------|-----------|--------------------------|
| HOT PEPPER Beauty | ¥25,000〜¥200,000 | ¥3,000〜¥8,000 |
| Instagram広告 | ¥50,000〜¥300,000 | ¥2,000〜¥5,000 |
| Google広告 | ¥30,000〜¥200,000 | ¥3,000〜¥10,000 |
| チラシ・ポスティング | ¥50,000〜¥150,000 | ¥5,000〜¥15,000 |
| 紹介キャンペーン | ¥10,000〜¥50,000 | ¥1,000〜¥3,000 |

**課題**:
- HOT PEPPER依存度が高く、掲載費が年々上昇
- クーポン目当ての「ジプシー客」が多く、LTVが低い
- チャネル別ROIの測定ができていない（効果不明のまま出稿）
- SNS運用のコスト（撮影・投稿・返信）が見えないコスト化

**ML解決の方向性**:
- チャネル別LTV予測（高LTV顧客を生むチャネルの特定）
- 広告投資最適化（ROI最大化のための配分モデル）
- 紹介確率予測（紹介しやすい顧客の特定）

### 4.3 人手不足・生産性

**ペインの深さ: ★★★★☆**

```
美容師の労働環境（業界平均）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
平均年収:           約330万円（全産業平均443万円の75%）
離職率（1年目）:    約50%
離職率（3年以内）:  約70%
有効求人倍率:       約3.5倍
平均労働時間:       月200時間以上（繁忙期220時間超）
施術以外の業務割合:  30〜40%（清掃、受付、在庫管理等）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**課題**:
- 高い離職率で採用・教育コストが膨大
- 施術以外の業務（予約管理、カルテ記入、在庫確認）が時間を圧迫
- 繁閑差が大きく、スタッフ配置の最適化が困難
- ベテランの「暗黙知」が共有されない

**ML解決の方向性**:
- 時間帯別来客数予測 → 最適シフト自動生成
- スタイリスト×メニュー別の生産性分析
- 教育効果予測（育成投資のROI可視化）

### 4.4 パーソナライズ対応

**ペインの深さ: ★★★★☆**

| 課題 | 詳細 | 影響 |
|------|------|------|
| カルテのデジタル化遅れ | 紙カルテが40%、検索・分析不可 | 顧客理解の限界 |
| 担当者依存 | スタイリスト退職で顧客情報消失 | 失客リスク |
| 提案の属人化 | 経験則ベースのメニュー提案 | 客単価の停滞 |
| 嗜好変化の未追跡 | 顧客のライフステージ変化に対応不可 | ミスマッチ |

**ML解決の方向性**:
- 顧客クラスタリング（施術履歴・来店パターン・単価帯）
- メニュー推奨エンジン（類似顧客の選択傾向から提案）
- 客単価アップ予測（追加メニュー受容確率の算出）

### 4.5 売上予測・経営分析

**ペインの深さ: ★★★☆☆**

**現状**:
- 月次売上は「前年同月比」での感覚的判断が主流
- 天候・イベント・季節変動の定量的分析ができていない
- 多店舗チェーンでも店舗間ベンチマークが手動Excel
- キャッシュフロー予測の精度が低く、設備投資判断が遅れる

**データの特性**:
- 美容室の売上は**季節性が非常に強い**（12月・3月・7月がピーク）
- 天候の影響大（雨天で予約キャンセル率が20〜30%上昇）
- 地域イベント・祝日の影響が顕著
- LightGBM回帰が最も直接的に適用可能な領域

**ML解決の方向性**:
- 日次/週次売上予測（天候・カレンダー特徴量含む）
- 店舗別ベンチマーク・異常検知
- 新店舗の売上シミュレーション

- 予約・POS・カルテが別システムで顧客IDの統合ができない

- 小規模店はスマホアプリの無料枠で運用しており、データ出力不可

**ML解決の方向性**:
- CSVインポート型のシンプルなデータ取込（技術障壁を最小化）

- ダッシュボード型のノーコード分析環境

### 6.1 離反予測AI

**優先度: ★★★★★（最優先）**

| 項目 | 内容 |
|------|------|
| **課題** | 顧客の離反を事前検知できず、手遅れになってから気づく |

| **出力** | 離反リスクスコア（0〜100）、予測次回来店日、推奨アクション |
| **手法** | LightGBM回帰（次回来店日数予測）+ 閾値による離反判定 |
| **精度目標** | 離反予測精度80%以上（30日超過の未来店を離反と定義） |

```python
# 特徴量例
features = [
```

## 8. 当社ポジショニング・差別化

### 8.1 ポジショニングステートメント

> **「既存の予約・POSシステムでは見えない"未来"を予測する、美容チェーン専用のAI分析プラットフォーム」**

既存SaaS（SALON BOARD, BEAUTY POS等）が「過去の記録・管理」に留まるのに対し、当社は**「未来の予測・最適化」**という上位レイヤーに特化。既存ツールと競合せず、補完関係を構築する。

### 8.2 差別化の3本柱

```
差別化戦略
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① 予測分析（Predictive Analytics）
   └ 離反予測・売上予測・来店予測
   └ 競合SaaSにはない唯一の機能
   └ LightGBM + SHAP による説明可能AI

② 多店舗横断分析（Cross-Store Analytics）
   └ 店舗間ベンチマーク・異常検知
   └ チェーン本部の意思決定支援
   └ 個店最適 → 全体最適への転換

③ アクション提案（Actionable Insights）
   └ 逆SHAPエンジンによる改善提案
   └ 「何を変えれば売上が上がるか」を提示
   └ データサイエンティスト不要で運用可能
```

## 14. 結論・推奨アクション

### 14.1 セグメント評価の結論

美容業界は**Phase 2での参入推奨セグメント**（スコア3.7/5.0）。

**参入すべき理由**:
1. **巨大な市場**: 25.8万店舗、ML分析の浸透率は5%未満 → ブルーオーシャン
2. **深いペイン**: 顧客離反が経営に直結、新規獲得コストは維持の5〜7倍
3. **技術適合性**: LightGBM回帰が来店予測・売上予測に直接適用可能
4. **既存SaaSとの補完**: 予約・POSツールの「上位レイヤー」として共存可能
5. **補助金活用**: IT導入補助金で初期コスト1/2〜3/4に圧縮

**注意すべき点**:
1. **データ取得の壁**: 最大のハードル、段階的アプローチで対応
2. **ITリテラシー格差**: UI/UXの徹底的な簡素化が必須
3. **価格感度**: 他業界比で低い → 成果保証・補助金の活用が鍵
