## 1. エグゼクティブサマリー

**食品製造業は、当社マルチアプリ・ポートフォリオ戦略において最も有望なセグメントである。**

| 項目 | 評価 |
|------|------|
| 市場規模 | 国内食品製造業 約25兆円（出荷額）、事業所数 約25,000 |
| ターゲット企業数 | 従業員50〜500名の中堅食品メーカー 約3,000〜5,000社 |
| ML/DS関連ペインの深刻度 | ★★★★★（需要予測・食品ロス・品質管理・人手不足が同時発生） |
| IT投資意欲 | ★★★★☆（DX補助金後押しもあり中堅企業で加速） |
| 当社ソリューションとの適合度 | ★★★★★（回帰予測・分類・異常検知が直接的にペインに対応） |
| 競合強度 | ★★★☆☆（大手SIer/コンサルは大企業向け、中堅向けは空白地帯） |
| 想定ARPU | 月額 8万〜25万円（ベース＋アプリアンロック） |
| 参入優先度 | **#1（最優先）** |

**選定理由のサマリー**:
- 需要予測・食品ロス削減は年間数千万円規模の経済インパクトがあり、投資対効果を示しやすい
- 品質管理（外観検査・異常検知）は食品安全法制の強化により「やらざるを得ない」需要
- 中堅企業はDX人材不在のため「使いやすいツール＋伴走支援」を強く求めている
- 当社のLightGBM回帰予測アプリが需要予測・歩留まり予測に直接活用可能
- ものづくり補助金（最大1,250万円）の活用により顧客の初期投資障壁を大幅に低減可能

## 2. 食品製造業の概況

### 2.1 市場規模と構造

| 指標 | 数値 | 出典 |
|------|------|------|
| 製造品出荷額 | 約25兆円 | 経済産業省 工業統計 |
| 事業所数 | 約25,000 | 経済産業省 工業統計 |
| 従業者数 | 約120万人 | 厚生労働省 労働力調査 |
| GDP構成比 | 製造業全体の約10% | 内閣府 |
| AI in Food市場（グローバル） | USD 123億（2024年）→ USD 352億（2032年予測） | Credence Research |
| AI in Food市場 CAGR | 14%（2024-2032） | Credence Research |
| 日本の食品加工ロボティクス市場 | USD 13億 | Ken Research |

### 2.2 業界の構造的特徴

1. **多品種・小ロット化の進行**: 消費者嗜好の多様化により、SKU数が増加し続けている
2. **賞味期限制約**: 生鮮品・日配品は1〜数日の賞味期限、在庫バッファが取れない
3. **季節変動の大きさ**: 気温・行事・天候により需要が大きく変動
4. **規制の厳格さ**: 食品衛生法、HACCP義務化、食品表示法などのコンプライアンス要件
5. **労働集約型**: 特に検査・仕分け・パッキング工程で人手に依存
6. **サプライチェーンの複雑さ**: 原材料調達→加工→物流→小売の多段階構造
7. **利益率の低さ**: 営業利益率 2〜5%が一般的で、コスト削減圧力が強い

### 2.3 DX化の現状

- 国内製造業の約87%がAIパイロットプロジェクトを開始（2025年時点）
- ただし食品製造業はDX化が「遅れている」とされる業種の一つ
- 最大の障壁は「導入コストの高さ」「DX人材の不足」「データ基盤の未整備」
- 50.8%の中小企業がDX年間投資を100万円未満に抑えている
- 一方、政府のDX促進補助金・ものづくり補助金が追い風となっている

## 3. ターゲット企業群の定義

### 3.1 プライマリターゲット（最優先）

**従業員50〜500名の中堅食品メーカー（推定 約3,000〜5,000社）**

| 条件 | 詳細 |
|------|------|
| 売上規模 | 10億〜500億円 |
| 従業員数 | 50〜500名 |
| 製造形態 | 自社工場を持つメーカー |
| IT部門 | 専任IT部門は小規模（0〜5名）、DX専任者はいないことが多い |
| IT予算 | 年間 500万〜3,000万円 |
| DXステージ | PoCまたは部分導入段階 |

**代表的なサブセグメント**:

| サブセグメント | 特徴 | ペインの特徴 |
|---------------|------|-------------|
| 冷凍食品メーカー | 多品種・需要変動大 | 需要予測、在庫最適化 |
| 水産加工 | 原材料価格変動、鮮度管理 | 歩留まり改善、品質予測 |
| 菓子・パン | 季節商品多数、短賞味期限 | 需要予測、廃棄ロス |
| 調味料・ソース | 比較的安定需要、品質管理重視 | 品質予測、原価管理 |
| 豆腐・日配品 | 賞味期限1〜3日、日次予測必須 | 需要予測（日次精度） |
| 食肉加工 | 原材料単価変動、歩留まり重要 | 歩留まり予測、原価分析 |
| 惣菜・弁当 | 超短賞味期限、天候依存 | 需要予測、食品ロス |

### 3.2 セカンダリターゲット

**食品製造業向けのサービス企業**:
- 食品卸売業者（需要予測・在庫最適化ニーズ）
- 食品物流企業（配送最適化ニーズ）
- 食品検査受託企業（検査効率化ニーズ）

### 3.3 ターゲットの地理的分布

食品製造業は全国に分散しているが、特に以下の地域に集中:
- **北海道**: 乳製品、水産加工
- **東北**: 米菓、水産加工
- **関東**: 総合食品メーカー、冷凍食品
- **中部**: 調味料、菓子
- **関西**: 菓子、即席食品
- **九州**: 水産加工、焼酎・酒類

→ リモート対応＋オンラインツール提供により、全国のターゲットにアプローチ可能

## 4. ペイン・課題の詳細分析

### 4.1 需要予測・食品ロス

**深刻度: ★★★★★ | 経済インパクト: 年間売上の2〜5%**

**現状の課題**:
- 食品ロスは日本全体で年間約523万トン（2021年度）、うち食品製造業が約121万トン
- 賞味期限制約により、需要予測の精度が収益に直結する
- 多くの中堅企業では、担当者の経験と勘に基づく「人力予測」が主流
- 天候・曜日・イベント・販促等の複合的な変動要因に対応できていない
- 欠品による機会損失と、過剰生産による廃棄ロスの両方が発生

**経済的インパクト（売上100億円企業の場合）**:
- 食品廃棄コスト: 年間 2,000万〜5,000万円
- 欠品による機会損失: 年間 1,000万〜3,000万円
- AI需要予測導入で20〜50%の改善が見込まれる → **年間 1,500万〜4,000万円の効果**

**成功事例**:
- 相模屋食料（豆腐）: AI需要予測で注文との誤差ほぼゼロ、廃棄ロス3割減、**年間1,000万円削減**
- ニチレイフーズ: AI需要予測で欠品・廃棄の両方を削減

### 4.2 品質管理・外観検査

**深刻度: ★★★★★ | 経済インパクト: 検査コスト30〜50%削減**

**現状の課題**:
- HACCP義務化（2021年〜）により全食品事業者に衛生管理の「見える化」が必要
- 外観検査は目視に依存、熟練検査員の疲労・バラツキが品質リスクに
- 不良品の流出は即リコール → ブランド毀損（1件あたり数千万〜数億円の損害）
- 検査員の確保が困難（人手不足の影響が最も大きい工程の一つ）
- 食品衛生法の改正で、トレーサビリティ強化も求められている

**経済的インパクト**:
- 目視検査要員: 1ライン2〜3名 × 年間人件費 500万円/人 = 1,000〜1,500万円/ライン
- AI検査で人員50%削減 + 検出精度向上 → **年間500万〜750万円/ライン削減**
- リコール防止の保険的価値: プライスレス（1件で数億円の損害回避）

**成功事例**:
- キユーピー: AI外観検査で検査速度2倍、従来の1/10のコストで導入
- 教師なし学習で100万個以上の良品画像を学習、異常品を自動検出

### 4.3 人手不足・技能伝承

**深刻度: ★★★★☆ | 経済インパクト: 生産性10〜30%向上**

**現状の課題**:
- 食品製造業の有効求人倍率は製造業平均を上回る高水準
- 特に地方の中堅企業では深刻な採用難
- ベテラン職人の経験・勘に依存する「匠の技」のブラックボックス化
- 退職・高齢化による技能喪失リスク（10年以内に熟練者の大量退職）
- パート・アルバイト比率が高く、品質のバラツキが生じやすい

**具体的な問題領域**:
- 調理工程の温度・時間管理（職人の感覚に依存）
- 原材料の品質判定（目利き技術）
- 設備のメンテナンス判断（異音・振動の感知）
- 配合調整（季節・原材料ロットによる微調整）

### 4.4 データ基盤の未整備

**深刻度: ★★★★☆ | 障壁としての影響度: 高**

**現状の課題**:
- 生産データが紙ベース・Excel管理の企業が依然として多い
- 複数システム（生産管理・在庫管理・販売管理）が別々に稼働し、データが分断
- IoTセンサーデータの収集・蓄積基盤がない
- データの品質が低い（欠損値、手入力ミス、フォーマット不統一）
- 「何のデータを取るべきか」すらわからない企業が多い

**当社にとっての意味**:
- **これは課題であると同時に大きなビジネスチャンス**
- データ基盤の整備支援からML活用まで一貫して支援できる
- 「データの整備→可視化→予測」のステップを段階的に提供可能

### 4.5 サプライチェーン最適化

**深刻度: ★★★☆☆ | 経済インパクト: 物流コスト10〜20%削減**

**現状の課題**:
- 原材料調達→加工→出荷→卸→小売の各段階でデータが分断
- サプライチェーン全体での需給最適化ができていない
- 原材料価格の変動（特に輸入原材料）への対応が後手に回る
- 2024年問題（物流ドライバー不足）の影響で配送効率化が急務
- 小売のPOSデータとメーカーの生産データの連携不足

### 4.6 原価管理・コスト最適化

**深刻度: ★★★★☆ | 経済インパクト: 利益率1〜3ポイント改善**

**現状の課題**:
- 食品製造業の営業利益率は2〜5%と低く、コスト管理が命綱
- 原材料費の変動（小麦、油脂、砂糖等の国際相場連動）に対する予測が不十分
- 歩留まりの変動要因（原材料品質、温度、湿度等）が定量化されていない
- エネルギーコスト上昇（電気・ガス）への対応も課題
- 多品種生産時のライン切替コストの最適化ができていない

**経済的インパクト（売上100億円企業の場合）**:
- 歩留まり1%改善 → 原材料費 年間 約1,000万円削減
- エネルギー使用量10%削減 → 年間 約500万円削減
- ライン稼働率5%改善 → 年間 約800万円の生産性向上

## 5. 市場価格・相場の調査

### 5.1 コンサルティング・アドバイザリー相場

| サービス形態 | 価格帯 | 内容 |
|-------------|--------|------|
| DXコンサル（アドバイザリー型） | 150万円〜/プロジェクト | DX戦略策定、ロードマップ作成 |
| AI導入コンサルティング（構想フェーズ） | 40万〜200万円 | ヒアリング、課題整理、AI活用提案 |
| 製造業向けコンサル（時間単価） | 5,000円〜10万円/時間 | 経験・スキルにより大幅に変動 |
| ITコンサル（導入支援型） | 月額50万〜100万円 | 戦略策定から導入支援まで一貫 |
| ITコンサル（ソフト導入のみ） | 月額10万〜30万円 | 特定ソフトウェア導入 |
| 成果報酬型 | 成果の10〜40% | 効果連動型、リスク共有 |

**当社へのインプリケーション**:
- 中堅食品メーカーにとって「月額50万〜100万円のITコンサル」は高額感がある
- **月額8万〜25万円のSaaS＋伴走支援**は「ちょうどいい価格帯」に位置
- 構想フェーズのコンサル（40万〜200万円）と比較して、ツール提供＋運用支援はROIが明確

- **当社はSaaS型で初期費用を大幅に低減（月額制で参入障壁を下げる）**
- 従来6ヶ月かかるプロジェクトを「即利用可能なツール」として提供 → 圧倒的な時間価値

### 5.5 補助金・助成金の活用

| 補助金名 | 上限額 | 補助率 | 対象 |
|---------|--------|--------|------|

- **導入**: AI需要予測システム
- **成果**: 注文との誤差ほぼゼロ（0.06%）、廃棄ロス3割減、**年間1,000万円削減**
- **示唆**: 短賞味期限商品こそAI需要予測の効果が大きい

### 13.2 キユーピー（マヨネーズ・離乳食）
- **課題**: 離乳食の外観検査に大量の検査員が必要
- **導入**: AI外観検査システム（教師なし学習、100万個以上の良品画像）
- **成果**: 検査速度2倍、従来の**1/10のコスト**で導入、品質管理の属人化解消
- **示唆**: 少ない初期投資でも高精度な品質検査AIは実現可能

### 13.3 ニチレイフーズ（冷凍食品）
- **課題**: 冷凍食品の需要は天候・季節で大きく変動
- **導入**: AI需要予測で生産計画を最適化
- **成果**: 欠品と廃棄の両方を削減、サプライチェーン全体の効率向上
- **示唆**: 冷凍食品は在庫バッファが取れる分、需要予測精度の向上が直接利益に

### 13.4 マヨネーズ・ドレッシングメーカー（非公開）
- **課題**: 販売データと季節要因から需要を正確に予測したい
- **導入**: AI需要予測（過去販売データ＋季節要因＋イベント情報）
- **成果**: フードロス10%削減、持続可能な生産体制構築
- **示唆**: 調味料カテゴリでも需要予測AIは有効
