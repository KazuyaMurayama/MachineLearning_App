## 1. エグゼクティブサマリー

### 士業×ML予測アプリの機会

**市場規模**: 日本のリーガルテック市場は$1.7B（2025年）→ $3.0B（2035年）、CAGR 5.7%。会計ソフト市場は$0.83B（2025年）。税務会計ソフト市場は$536M（2024年）→ $1,910M（2035年）、CAGR 12.24%。Legal AI Software市場はグローバルで$5.21B（2026年）→ $40.94B（2034年）、CAGR 29.4%と急成長。

**セグメント評価スコア: 3.6 / 5.0**（Phase 2-3推奨）

| 評価項目 | スコア | 根拠 |
|---------|--------|------|
| 市場規模・成長性 | ★★★★☆ | 税理士8万人・弁護士4.5万人、DX需要急増 |
| ペインの深さ | ★★★★☆ | 顧問先離反・価格競争・AI代替不安が深刻 |
| 技術適合性 | ★★★★☆ | LightGBM回帰が顧問料予測・解約予測に適用可 |
| 競合状況 | ★★★★☆ | 業務SaaSは充実、ML経営分析は空白 |
| 導入障壁 | ★★★☆☆ | ITリテラシーは高いがデータ活用意識に差 |
| 単価ポテンシャル | ★★★☆☆ | 中規模法人で月額5〜15万円が現実的 |

**核心的インサイト**:
- 士業の最大ペインは**顧問先の離反・低価格競争**（クラウド会計普及で記帳代行の価値が低下）
- 既存SaaS（freee, MF, 弥生等）は会計処理の効率化が主で**経営分析・予測機能が未成熟**
- 生成AIの台頭で「AIに代替される」不安が業界全体に蔓延 → AI活用への投資意欲が高まっている
- 約62%の法律事務所が自動化ツールを導入、58%がAIを文書レビューに活用（グローバル）

**推奨戦略**: 中規模税理士法人・会計事務所（職員10名以上）をメインターゲットに、顧問先離反予測AI + 顧問先別LTV分析をコアとしたスイート提供。「AIに仕事を奪われる」側ではなく「AIを武器にする」側への転換を支援するポジショニング。

## 3. ターゲット企業群の定義

### 3.1 セグメンテーション（主に税理士・会計事務所を軸に）

| セグメント | 職員数 | 年商規模 | 推定数 | 優先度 |
|-----------|--------|---------|--------|--------|
| **Tier 1: 大手税理士法人** | 50名以上 | 5億円以上 | 約100法人 | ★★★★★ |
| **Tier 2: 中規模法人** | 10〜49名 | 1〜5億円 | 約800法人 | ★★★★★ |

| **Tier 4: 個人事務所** | 1〜2名 | 〜3,000万円 | 約28,000事務所 | ★★☆☆☆ |

### 3.2 メインターゲット：Tier 1-2（中大規模税理士法人）

**選定理由**:
- **データ量**: 数百〜数千の顧問先データ、月次試算表・決算書の蓄積が豊富
- **投資余力**: 月額5〜20万円のIT投資が可能な財務基盤
- **経営課題の顕在化**: 顧問先離反・職員離職・付加価値化が経営課題
- **意思決定者**: 代表税理士が経営者意識を持ち、ROIベースで判断
- **クラウド会計導入済み**: freee/MFパートナーとしてデータ連携の下地あり

**代表的な法人例**:
- 辻・本郷税理士法人（約2,000名）
- 税理士法人山田&パートナーズ（約600名）
- 各地域の30〜100名規模法人

### 3.3 サブターゲット：弁護士事務所・社労士事務所

| 士業カテゴリ | メインターゲット | 主なペイン | ML適用可能性 |

## 4. ペイン・課題の詳細分析

### 4.1 顧問先の離反・解約

**ペインの深さ: ★★★★★（最重要）**

| 指標 | 数値 | インパクト |
|------|------|-----------|
| 年間顧問先離反率 | 5〜10% | 100社の事務所で年5〜10社が離脱 |
| 離反1社あたりの損失 | ¥300,000〜¥1,200,000/年 | 顧問料+決算料の逸失 |
| 新規顧問先獲得コスト | ¥100,000〜¥500,000/件 | Web広告・紹介・セミナー等 |
| 離反理由トップ3 | 価格/対応品質/後継者不在 | 事前に予兆を掴めない |

**現状の対応**:
- 担当者が「なんとなく」危ないと感じた顧問先を所長に報告
- 年1回の顧客満足度調査（回答率が低い）
- 離反してから気づくケースが大半
- 価格交渉が来てから対応する受動的な姿勢

**ML解決の方向性**:
- 顧問先の行動データ（連絡頻度・質問数・面談実績・入金タイミング）からの**離反リスクスコアリング**
- LightGBM回帰で「解約までの予測月数」を算出
- リスクスコアに応じた先制フォロー施策（面談設定・付加価値提案等）

### 4.2 業務効率化・生産性

**ペインの深さ: ★★★★★**

```
税理士事務所の業務時間配分（典型的な月次サイクル）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
記帳・仕訳入力:    ████████████░░░░░░░░  30%  ← AI-OCR/クラウド化で削減可能
月次チェック:      ██████████░░░░░░░░░░  25%  ← 自動チェックで効率化可能
顧問先対応:        ████████░░░░░░░░░░░░  20%  ← 高付加価値業務
税務判断・相談:    ████░░░░░░░░░░░░░░░░  12%  ← コア業務
事務作業:          ████░░░░░░░░░░░░░░░░  8%
営業・マーケ:      ██░░░░░░░░░░░░░░░░░░  5%   ← 最も不足
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**課題**:
- 記帳代行の単純作業が全体の30%を占め、高付加価値業務に時間を割けない
- 繁忙期（確定申告2〜3月、決算期集中月）に業務が集中し、残業が常態化
- 顧問先ごとの工数管理ができておらず、不採算先の特定が困難
- 「忙しいのに儲からない」状態が慢性化

**ML解決の方向性**:
- 顧問先別の予測工数算出 → 不採算顧問先の特定
- 繁忙期の業務量予測 → 人員計画の最適化
- 業務パターン分析 → ボトルネックの特定と改善提案

### 4.3 付加価値サービスの不足

**ペインの深さ: ★★★★☆**

| 課題 | 詳細 | 影響 |
|------|------|------|
| 記帳代行の価値低下 | クラウド会計で自動仕訳が可能に | 顧問料の値下げ圧力 |
| 経営助言の属人化 | ベテラン税理士の経験則頼み | スケールしない |
| データ分析力の不足 | 試算表を「作る」が「読み解く」が弱い | 差別化不可 |
| コンサル転換の遅れ | 「作業」から「助言」への転換が進まない | 顧問先の期待値とのギャップ |

**顧問先が本当に求めていること**:
```
顧問先のニーズ優先度（船井総研調査等を参考）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1位: 経営数値の見える化・わかりやすい説明
2位: 節税アドバイス・税務リスクの事前警告
3位: 資金繰り予測・キャッシュフロー管理
4位: 経営改善の具体的提案
5位: 補助金・助成金の情報提供
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
→ すべてデータ分析・予測に基づくサービス！
```

**ML解決の方向性**:
- 顧問先の財務データからの**経営リスク予測**（倒産リスク・資金ショートリスク）
- SHAP解釈による「なぜこの数値が危ないか」の自動説明生成
- 同業他社ベンチマーク（匿名化データでの比較分析）

### 4.4 人材不足・採用難

**ペインの深さ: ★★★★☆**

| 指標 | 数値 | インパクト |
|------|------|-----------|
| 税理士事務所の有効求人倍率 | 約3〜5倍 | 採用が困難 |
| 職員離職率 | 年15〜20% | 5年で全職員が入れ替わるペース |
| 平均年収（税理士事務所職員） | 約350万円 | 他業界との競争力低 |
| 教育期間 | 2〜3年で一人前 | 育成コストが高い |

**課題**:
- 繁忙期の長時間労働が離職の主因
- クラウド会計の普及で「記帳屋」のイメージが強まり、優秀な人材が集まりにくい
- ベテラン職員の退職で顧問先対応品質が低下 → 顧問先離反の連鎖
- パート・アルバイトの活用が進まない（専門性の壁）

**ML解決の方向性**:
- 職員別の業務負荷分析 → 過負荷の早期検知
- 離職リスク予測（残業時間・担当件数・繁忙度パターン）
- 適正な顧問先配分の最適化

### 4.5 売上予測・事務所経営

**ペインの深さ: ★★★☆☆**

**現状**:
- 月次売上は「顧問料×社数+スポット業務」のシンプルな構造だが予測していない
- 決算月の集中度によって月間売上が2〜3倍変動
- 新規獲得・離反の予測ができず、年間計画が「昨年と同じ」で策定
- スポット業務（相続税申告、事業承継、M&A等）の発生が読めない

**士業特有のデータ特性**:
- 顧問料は月額固定のため、ストック型で読みやすい部分がある
- 決算月の偏り（3月・12月決算が集中）
- 確定申告期（2〜3月）は年1回の最大繁忙期
- スポット業務は高単価だが発生タイミングが不定

**ML解決の方向性**:
- 月次/四半期売上予測（ストック+スポットの二層構造）
- 顧問先の決算月分布と業務量予測
- 新規獲得チャネル別の効果予測

### 4.6 データ基盤・デジタル化の遅れ

**ペインの深さ: ★★★☆☆**

```
士業のデータ分断状況
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  会計ソフト       税務ソフト       顧問先管理
  (freee/MF/弥生)  (達人/魔法陣)    (Excel/独自)
       │              │              │
       └──── 一部連携 ────┘              │
                  │                     │
                  └──── 連携なし ────────┘
                         │
              ┌──────────┴──────────┐
              │ 経営分析は手動集計   │
              │ → 顧問先情報が散在  │
              └─────────────────────┘
```

**士業が他セグメントより有利な点**:
- 会計データ（試算表・決算書）は構造化されており、ML学習に適している
- freee/MFのAPI連携が可能 → データ取得の自動化がしやすい

### 6.1 顧問先離反予測AI

**優先度: ★★★★★（最優先）**

| **精度目標** | 離反予測精度75%以上（6ヶ月以内の解約を予測） |
| **ビジネス価値** | 離反率3%改善 → 年間¥300〜600万の逸失売上回復（100社規模事務所） |

# 特徴量例
