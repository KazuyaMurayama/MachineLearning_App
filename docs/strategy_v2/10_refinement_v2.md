```
■ M12以降の月間稼働（AI自動化なし vs あり）

活動                AI自動化なし    AI自動化あり    節約
─────────────────────────────────────────────────
顧客対応（10社）     10.0h          10.0h          0h（電話は削れない）
```

**効果:**
- AI自動化なしでは10社で39h → **32h超過で破綻**

### 節約された14.5hの推奨配分

### 4.4 ボトムアップ再検証

#### 検証①: 「3週間に1社」の営業は月32hで可能か？

1社の新規獲得に必要な活動を時間分解:

```
■ 1社の営業プロセス（成約まで）

  リード発掘・初回アプローチ:     2-3h（複数社にアプローチ）
  初回ヒアリング（オンライン）:   1h
  PoC提案書作成:                  2-3h
  PoC実施（データ受領→分析）:    4-6h
  結果報告＆商談:                 1-2h
  契約・オンボーディング:        3-5h
  ──────────────────────────────
  合計: 13-20h/社（中央値16h）

  成約率: 30-50%（PoC→有償）
  → 1社成約に必要な総営業投資: 16h ÷ 40% = 約40h
```

**⚠️ 発見:** 月0.7社の新規獲得には月28h相当の営業投資が必要（40h × 0.7 = 28h）。**これは月32hの87.5%を営業だけで消費する。**

**しかし現実には:**
- M1-M3は営業なし（開発中心）
- M4-M12は既存顧客対応と営業の並行が必要
- 紹介経由の顧客は営業工数が60%減（16h→6h）
- 診断士提携後の紹介は商談から開始（リード発掘不要）

**修正後の営業モデル:**

| 獲得チャネル | 構成比 | 営業工数/社 | 加重工数 |
|------------|--------|-----------|---------|
| 直接営業 | 30% | 16h | 4.8h |
| 診断士紹介 | 40% | 8h | 3.2h |
| SEO/コンテンツ経由 | 20% | 10h | 2.0h |
| 既存顧客紹介 | 10% | 5h | 0.5h |
| **加重平均** | **100%** | | **10.5h/社** |

月0.65社 × 10.5h = **月6.8h**の営業投資。これなら32hに収まる。

**ただし条件付き:** 紹介チャネルが機能するM6以降の数値。M4-M5は直接営業中心のため月1社は困難。

#### 検証②: 10社のサポートは月10hで済むか？

製造業顧客のサポート実態を分解:

```
■ 顧客あたりの月間サポート時間（分解）

  月1回の定例電話:         15分（これは削れない）
  データ品質の問い合わせ:   10-30分（月によるばらつき大）
  予測結果への質問:         10-15分
```

### 4.6 第3独立検証エージェントの超保守的推定

第3の独立検証エージェントは、§9の数値に対し**極めて厳しい**判定を下した。

```

  「§9の標準シナリオ¥1,700万は、全変数が標準以上で推移する前提。
```
