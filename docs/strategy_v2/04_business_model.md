## 4.1 ユニットエコノミクスの精密設計

### CAC（顧客獲得コスト）業界別推計

#### 最適チャネルミックス

**製造業（ブレンドCAC: ¥82,881）**

| チャネル | 構成比 | 単体CAC | 加重CAC |
|---------|--------|---------|---------|
| トップダウンPLG | 30% | ¥16,667 | ¥5,000 |
| コンテンツMKT | 20% | ¥100,000 | ¥20,000 |
| ウェビナー | 10% | ¥91,429 | ¥9,143 |
| SIerパートナー | 35% | ¥0（※） | ¥0 |
| 展示会 | 5% | ¥750,000 | ¥37,500 |

※パートナー経由は初期CACゼロ、レベニューシェア30%で継続支払い

**物流業:** ブレンドCAC ¥95,000（IT投資決定プロセスが長い、平均3ヶ月）
**不動産業:** ブレンドCAC ¥65,000（店舗数が多くPLG向き、FC経由一括導入あり）

### LTV（顧客生涯価値）

| パラメータ | 製造業 | 物流業 | 不動産業 |
|-----------|--------|--------|---------|
| 月次ARPU | ¥45,000 | ¥50,000 | ¥28,000 |
| 粗利率 | 85% | 85% | 88% |
| 月次チャーン率 | 3.0% | 2.5% | 4.0% |
| 平均契約月数 | 33ヶ月 | 40ヶ月 | 25ヶ月 |
| **LTV** | **¥1,260,000** | **¥1,700,000** | **¥616,000** |

### LTV最大化施策

**ARPU向上（+¥15,000〜50,000/月）:**
- アップセル（Starter→Professional）: +¥15,000/月
- 追加モデル課金（¥10,000/モデル/月）: +¥10,000/月
- API連携課金: +¥20,000/月
- レポート自動生成プレミアム: +¥5,000/月

**チャーン率低減（-0.5〜2.0%）:**
- 自動再学習機能: -0.5%
- 精度モニタリングアラート: -0.3%
- 業界ベンチマーク比較: -0.4%
- 年間契約割引（20%OFF）: -0.8%

### 健全性指標

| 指標 | 製造業 | 物流業 | 不動産 | 目標 |
|------|--------|--------|--------|------|
| **LTV/CAC** | **15.2x** | **17.9x** | **9.5x** | >5x ✅ |
| **Payback** | **2.2ヶ月** | **2.2ヶ月** | **2.6ヶ月** | <12ヶ月 ✅ |

→ **全業界で極めて健全。マーケティング投資を積極的に増やすべき。**

## 4.2 収益モデルの革新

### 5つの収益ストリーム

```
┌─────────────────────────────────────────────────────┐
│                収益ストリーム構成                      │
│                                                       │
│  ① サブスク(60%)  ② 従量課金(20%)  ③ 成果報酬(10%) │
│  月額固定         予測回数×¥10     精度改善×成果%   │
│  ¥29,800〜        API呼出×¥5                        │
│                                                       │
│  ④ データ収益(5%)           ⑤ プロサービス(5%)      │
│  ベンチマークレポート販売    導入支援・カスタム開発   │
└─────────────────────────────────────────────────────┘
```

### 業界別ROI定量化

#### 製造業: 見積精度1%改善の経済効果

```
■ 現状コスト（月間）
  ベテラン担当者の時間:  月40h × ¥5,000/h  = ¥200,000
  見積ミスによる損失:    月2件 × ¥500,000   = ¥1,000,000
  合計:                                       ¥1,200,000

■ 導入後の削減効果
  見積時間 50%削減:                            ¥100,000
  見積ミス 70%削減:                            ¥700,000
  合計削減:                                    ¥800,000/月

■ 投資対効果
  月額投資:    ¥49,800
  月間削減:    ¥800,000
  ROI:         1,506%
  回収期間:    約2日
```

#### 物流業: 車両稼働率1%改善の経済効果

```
■ 現状コスト（月間）
  過剰車両配置:  月5台 × ¥300,000/台  = ¥1,500,000
  機会損失:      月3件 × ¥200,000     = ¥600,000
  合計:                                  ¥2,100,000

■ 導入後の削減効果
  過剰配置 60%削減:                      ¥900,000
  機会損失 50%削減:                      ¥300,000
  合計削減:                              ¥1,200,000/月

■ 投資対効果
  月額投資:    ¥49,800
  ROI:         2,309%
  回収期間:    約1.3日
```

#### 不動産業: 空室率1%低下の経済効果

```
■ 現状コスト（月間）
  空室損失:  10物件 × ¥80,000/物件     = ¥800,000
  属人的対策の人件費:                    = ¥200,000
  合計:                                   ¥1,000,000

■ 導入後の改善効果
  空室期間 30%短縮:                      ¥240,000
  対策効率化:                            ¥100,000
  合計改善:                              ¥340,000/月

■ 投資対効果
  月額投資:    ¥19,800
  ROI:         1,617%
  回収期間:    約1.8日
```
